## 三、付费分层方案（核心重点）

### 设计原则
> **免费给"体验"，收费给"体系"**
> 免费闭环：刷 + 听 + 查（让用户形成习惯）
> 付费闭环：练 + 评 + 复习 + 进阶（让用户获得成长）

### 功能对照表

| 功能 | 🆓 免费用户 | 👑 付费会员 |
|------|------------|------------|
| **刷视频** | ✅ 无限量 | — |
| **闪答** | ✅ 完全免费 | — |
| **纯英/中英/减压字幕** | ✅ 完全免费 | — |
| **延迟字幕** | ❌ | ✅ 可自定义延迟 2/3/5 秒 |
| **字典查词+框选** | ✅ 完全免费 | — |
| **收藏单词** | ✅ 不限量 | — |
| **AI 教练** | 每日 3 次 | ✅ 无限次 |
| **学习笔记** | 前 2 条（后面毛玻璃） | ✅ 完整内容 |
| **跟读（听+跟念）** | ✅ 免费 | — |
| **跟读 AI 评分** | ❌ | ✅ 评分+问题定位+建议 |
| **搜索→视频语境** | 每次展示 3 条 | ✅ 无限+难度筛选 |
| **难度筛选视频** | ❌ 跟算法推荐 | ✅ 自选1~5星 |
| **每日守护+复习系统** | ❌ | ✅ 遗忘曲线/统计/复习计划 |
| **微练习** | 每日 3 题 | ✅ 无限 |
| **重遇弹幕** | ✅ 不限次 | ✅ 点击后可加复习队列/加练/查周报 |
| **学习数据报告** | ❌ | ✅ 掌握率/连续天数/增长曲线 |

### 免费限制的体验细节（重要）

| 触发场景 | 免费用户看到的 |
|----------|--------------|
| AI 教练第4次点击 | AI 输出前两行内容，后面模糊化，显示"解锁完整解析" |
| 学习笔记第3条起 | 毛玻璃遮盖 |
| 微练习第4题起 | 题目正常展示，但选项模糊，显示"升级解锁，继续练习" |
| 单词即将降级时 | 提示"你有N个单词即将降级，开启每日守护保护你的词汇" |

## 七、2026-04-27 周一会议 — 微练习 & 单词体系新决策

### 已确定事项

#### 1. 微练习与熟悉度体系打通
- 微练习出的题目包含用户收藏的单词
- 答题结果直接推进/影响该单词的熟悉度等级
- 微练习不再是独立模块，而是熟悉度提升的核心手段之一

#### 2. 导入辞书支持过滤 A1/A2 单词
- 用户导入整本辞书时，可一键过滤掉 A1、A2 级别的基础单词
- 只保留有学习价值的词进入待学习队列
- 减少用户面对几千生词的心理压力

#### 3. 收藏夹新增「学习列表」
- 用户可设置每月学习目标（本月要学多少个单词）
- 系统自动从待学习队列填充到学习列表
- 目标：将学习列表中的单词全部推到「精通」
- 当月未完成的单词累计到下个月
- 用户可随时调整本月学习目标数量
- 学习列表中的单词通过微练习、每日守护、考一下等方式推进

### 讨论背景

- 原方案一个视频只出一题，老板希望增加到最多 6~7 题
- 讨论结论：题目数量应与视频内容信息密度匹配，建议 3~5 题为主
- 单词债问题：用户长期不复习导致降级堆积 → 通过「学习列表」按月控制活跃词量，从源头避免债务无限膨胀
- 微练习计费方式：按视频 session 计（1 个视频的练习 = 1 次），不按题目数量计

### 补充决策（同日）

#### 4. 取消终身会员
- 暂不考虑终身会员方案
- 原定的 298~348 元限量 500 个方案取消

#### 5. 取消月付，只保留年付
- 理由：v0.6 功能粘性不足，月付用户容易用一个月就流失
- 教育产品走高客单价路线，不靠月付小额诱导
- 只保留年付，通过沉没成本倒逼用户坚持使用、养成习惯

#### 6. 定价调整
- 第一阶段年卡起步价：198 元（原 98~128）
- 第二阶段：299 元
- 第三阶段：398 元
- 内测专属价保持 68 元/年

#### 7. 创始会员起步价最终确定：299 元/年
- 最终阶梯：299 → 399 → 498
- 内测专属价保持 68 元/年
