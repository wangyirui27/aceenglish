# 15 秒对话分镜生成 Skill

## 使用场景

用户提供：
- 一个主题 / 一句话 / 一个情景（可能含编号、标签、核心句）
- CEFR 难度等级（A1 / A2 / B1 / B2 / C1 / C2）

产出（v7.0 · 中文 prompt 写满版）：
- **只交付一段中文 prompt**，字数目标 **1800-2000 字符**（贴着上限写，不要刻意精简）
- 💡 核心原理：即梦每次生成消耗的积分固定，不论 prompt 长短。既然成本一样，**把 2000 字预算用满**可以获得更丰富的细节（微表情、光影、材质、环境），成片质量更高
- 即梦是国产工具，对中文指令理解更好，且中文比英文省 60% 字符（同等信息量），所以用中文能塞更多细节
- **台词部分保留英文原文**（加双引号），因为台词是头号听力的教学核心，不能翻译
- 不要写基本信息表、角色设定章节、分镜概要、教学要点、自检清单等中文文档
- 所有信息（人物外貌、剧情、对话、镜头、光影、环境、材质）全部融进那一段中文 prompt 里
- 如果用户明确要求中英对照或教学要点，再单独追加，否则**只给 prompt 一段**

## 核心约束（不可违反）

### 1. 时长硬上限
- **总时长 ≤ 15 秒**，绝对不能超
- 时间戳精确到秒（如 0-3s, 3-7s, 7-11s, 11-15s）

### 2. 对话结构
- **一来一回 = 1 回合**
- 目标：**2 回合（4 句台词）**
- 降级条款：如果 15 秒装不下 4 句，降到**最少 3 句（1.5 回合）**
- 绝不少于 3 句

### 3. 台词难度标注
按 CEFR 等级标注每句台词：

| 等级 | 特征 | 句长参考 |
|------|------|---------|
| A1 | 最基础，简单现在时 | 3-6 词 |
| A2 | 常用动词，简单过去/将来 | 5-8 词 |
| B1 | 从句、情态动词 | 8-12 词 |
| B2 | 复杂时态、抽象表达 | 10-15 词 |
| C1 | 地道表达、委婉语气 | 12-18 词 |
| C2 | 俚语、文学性、双关 | 灵活 |

### 4. 题材禁区
**不要**：科幻、恐怖、惊悚、血腥、超自然
**可以**：日常、校园、职场、家庭、友情、爱情、喜剧、悬疑（非惊悚）、治愈、宠物、旅行、美食、体育、音乐、文艺等

### 4b. 品牌禁区（v7.2 新增 · 即梦版权过滤）
**绝对不能出现的真实品牌名**（会触发即梦"输入文本可能涉及版权限制"）：
- ❌ 手机/电脑：iPhone / MacBook / Google Maps / Samsung / Huawei
- ❌ 咖啡/餐饮：Starbucks / McDonald's / KFC
- ❌ 相机/设备：ARRI Alexa / Canon / Sony / Nikon / 富士
- ❌ 胶片型号：Kodak Vision3 250D / Kodak Portra 400 / Fuji Pro 400H
- ❌ 真实地名：纽约曼哈顿 / 洛杉矶 / 巴黎 / 东京（用"某欧美城市""某国际机场"替代）
- ❌ 影视IP：黑客帝国 / 复仇者联盟等

**替代方案**：
| ❌ 禁写 | ✅ 改写 |
|---------|---------|
| Starbucks外带咖啡纸杯 | 外带咖啡纸杯（杯身有手握指纹） |
| iPhone / 智能手机 | 手机（屏幕显示地图App） |
| MacBook | 笔记本电脑 |
| ARRI Alexa质感 | 电影级摄影机质感 |
| Kodak Vision3 250D风格 | 日光电影卷风格，细腻颗粒感 |
| 纽约曼哈顿 | 某欧美城市街角 |
| 洛杉矶国际机场 | 某国际机场 |
| 富士胶片相机 | 复古胶片相机 |

### 5. 提示词字数区间
- **目标区间**：1800-2000 字符（含空格和标点），贴着上限写
- **硬上限 2000 字符**，绝不能超
- **下限 1800 字符**：如果写少于 1800，说明还有细节没加满，继续扩充
- 交付前实际数字符数并标注
- 积分与 prompt 长度无关 → 不用省字，用满预算让画面更细
- 绝不砍：台词、核心动作、人物主要识别特征

### 6. 镜头结构
每 15 秒段落 **3-4 个镜头**（对话戏优先 4 镜头，如下）：
- 镜头 1：角色 A 开口（近景特写或过肩） → 核心句放这里（黄金前 3 秒）
- 镜头 2：角色 B 反应 + 回应（近景特写）
- 镜头 3：角色 A 第二次说话（可切景别或过肩）
- 镜头 4：角色 B 第二次回应（收束情绪的特写）

## 🔑 即梦提示词黄金格式（v7.0 · 中文写满 · 活人感框架）

**一整段中文 prompt，1800-2000 字符。台词保留英文原文。**

### 结构模板（按顺序）

```
AI生成真人风格对话短剧，非真人取材。[时长+场景+画幅]，电影级摄影机质感，[拍摄手法]，[胶片型号]。

光源：[光从哪来→打在什么位置→产生什么明暗效果]。

角色A：[外貌+服装+不完美细节+状态描述]。
角色B：[外貌+服装+不完美细节+状态描述]。

镜头1【Xs-Xs】[景别]+[运镜指令]。[动作链→面部反应→身体动作→情绪收尾]。[微表情肌肉级+情绪幅度百分比]。[物理细节：惯性/重力/风阻/光线]。[三层构图：前景+中景+背景]。[台词]: "[英文台词]"

镜头2...

镜头3...

镜头4...

环境：[环境元素1（叙事理由）]，[环境元素2（光影来源）]，[环境元素3]。[感官细节：声音/温度/触感]。[缺陷美学：晃动/灰尘/运动模糊]。[无字幕无水印无配乐]。

负面：[只排除该场景可能误出现的5-8项]。
```

### 七层填充指南

#### 第 1 层：开场（场景+画幅+设备+拍摄手法+胶片）
```
AI生成真人风格对话短剧，非真人取材。15秒双人对话，
[地点描述]，16:9横版，电影级摄影机质感，
手持拍摄镜头轻微晃动增加纪实感，日光电影卷日光电影卷拍摄，
自然颗粒感而非数码平滑。
```

**设备关键词**（比"摄影写实"有效 10 倍）：
| 中文写法 | 效果 |
|----------|------|
| 电影级摄影机质感 | 电影级色彩科学，暗部层次丰富 |
| 高端数码相机质感 | 高清晰度广告感 |
| 徕卡色调 | 温润复古，偏暖 |
| 蔡司镜头质感 | 锐利，微距细节出色 |

**拍摄手法关键词**（Seedance 2.0 识别率高）：
| 中文指令 | 效果 |
|----------|------|
| 手持拍摄，镜头轻微晃动 | 纪实感、呼吸感、紧张感 |
| 镜头缓缓推近 | 聚焦，紧张感/关注感 |
| 镜头缓缓拉远 | 揭示全貌，孤独感 |
| 镜头环绕主体旋转 | 360度展示，仪式感 |
| 镜头跟随主体移动 | 跟拍，动态叙事 |
| 长焦镜头，背景虚化突出主体 | 压缩空间，人物特写 |

#### 第 2 层：光源（光从哪来→打在什么位置→产生什么明暗效果）

**不要写色温参数，写物理效果。**
```
❌ 色温约5600K，侧光照入
✅ 午后自然侧光从左窗照入，在Leo面部形成清晰的明暗分割——
   右颧骨到下巴是柔和高光，左侧轮廓沉入温暖阴影，
   头顶荧光灯从上方在额头叠一层冷白高光，
   两种光源在下巴处交汇形成微妙的冷暖渐变。
```

**5 种经典光效模板**（按场景选）：

| 光效 | 写什么 | 适用 |
|------|--------|------|
| **侧逆光** | 侧逆光从[方向]打入，在人物[部位]勾勒出轮廓光 | 人像通用，塑造立体感 |
| **黄金时刻** | 日落前暖黄色柔和光线，人物皮肤通透，自带轮廓光 | 日常、户外、治愈 |
| **伦勃朗光** | 伦勃朗光，面部三角形光斑，光影衔接过渡自然 | 古典、神秘、人物刻画 |
| **窗边柔光** | 窗边柔光从左侧照入，面部柔和阴影过渡 | 日常、真实、室内对话 |
| **冷暖混合** | [暖光源]与[冷光源]在[部位]形成微妙冷暖渐变 | 室内混合光源场景 |

#### 第 3 层：角色（外貌+服装+材质状态化+不完美细节+情绪幅度）

每个角色要写：
1. **识别特征**（脸型+眼+发+服装主色+手上拿什么）
2. **材质状态**（反映角色当前状态，两个人绝不能写同一段）
3. **不完美细节**（至少 3 个，每个有叙事理由）
4. **三层肌理**（皮肤可见毛孔+原生纹理+自然瑕疵）

**三层肌理速查**（去 AI 感 3 关键词）：
| 类别 | 写什么 |
|------|--------|
| 基础肌理 | 可见毛孔、原生皮肤纹理、自然肤色不均 |
| 微瑕疵 | 轻微脸颊泛红、淡淡的原生小瑕疵、唇纹 |
| 动态细节 | 细微面部微表情、睫毛自然阴影、发丝动态光影 |
| 质感 | 皮肤自然油脂光泽（非塑料感） |

#### 第 4 层：动作链（情绪要有动作链条配合）

**核心原则：一个表情不够就加多个递进动作。**

动作链格式：**眼神变化 → 面部反应 → 身体动作 → 情绪收尾**

| 情绪 | 动作链 |
|------|--------|
| 求助 | 视线锁定目标 → 脚步减速 → 眉毛内侧上扬 → 嘴唇微启 → 身体前倾 |
| 犹豫 | 视线游移 → 轻咬下唇 → 眉头微蹙 → 手指无意识摆动 → 叹气 |
| 惊喜 | 眼睛睁大 → 嘴巴微张 → 嘴角上扬 → 双手抬起 → 身体前倾 |
| 愤怒 | 瞳孔收缩 → 下颌收紧 → 眉头紧锁 → 鼻翼张开 → 肩膀绷紧 |
| 坚定 | 直视前方 → 下颌收紧 → 嘴唇紧闭 → 肩膀后收 → 点头 |

**情绪幅度控制**（v7.0 新增）：
```
[情绪]，程度在[百分比]%，[附加性格描述]
```
- 求助，程度在40%，克制中带不安
- 友善微笑，程度在50%，自然不夸张
- 焦急，程度在60%，隐忍克制

#### 第 5 层：物理法则（运动要有阻力和重量感）

**核心：描述阻力之后，AI 才会计算出动作关系，画面才有重量感。**

| 物理要素 | 写什么 | 效果 |
|----------|--------|------|
| 风阻 | 发丝被微风拂向右侧，外套衣摆因走路惯性轻微摆动 | 运动有方向感 |
| 重力 | iPad从左手滑到腋下夹住有重力感，咖啡杯液体因转动轻微晃动 | 有重量感 |
| 惯性 | 急停时衣摆因惯性前摆，转头时发梢轻轻摆动 | 运动真实感 |
| 声音暗示 | 远处车流低频底噪，拉杆箱轮子在地砖上的滚动声 | 画面有声音感 |
| 温度感 | 呼吸在微凉空气中形成极淡白雾，鼻尖冻红 | 环境温度 |

#### 第 6 层：三层构图（前景遮挡+中景对焦+背景虚化）

**每条提示词至少一个镜头要写三层构图。**
```
前景有虚化的[物体]，中景[角色]清晰对焦，背景[环境]柔化
```

**前景遮挡物的叙事含义**：
| 前景物 | 叙事含义 |
|--------|----------|
| 消防栓/栏杆 | 都市日常，街道生活 |
| 绿植枝叶 | 自然、治愈、温暖 |
| 纱帘/窗框 | 偷窥、浪漫、私密 |
| 咖啡杯/食物 | 日常、社交、放松 |
| 书本/文件 | 学习、工作、思考 |

#### 第 7 层：缺陷美学 + 环境叙事

**缺陷 = 真实。每条提示词至少加 2 种缺陷。**

| 缺陷类型 | 写什么 |
|----------|--------|
| 镜头晃动 | 手持拍摄镜头轻微晃动增加纪实感 |
| 灰尘颗粒 | 空气中有极轻微光尘颗粒被阳光照亮 |
| 运动模糊 | 背景行人动态模糊，车辆运动模糊驶过 |
| 轻微暗角 | 画面边缘有极微暗角 |
| 色散 | 阳光直射处有极微色散 |

**环境叙事**：每个环境元素必须有**光影来源**或**叙事暗示**理由。

## 🎬 活人感框架（v7.0 新增 · 来自8篇新文档）

### A. 开场声明必须写设备+运镜
```
❌ 纪录片式自然摄影风格
✅ 电影级摄影机质感，手持拍摄镜头轻微晃动
```
**设备词比风格词有效10倍**。速查：

| 设备 | 效果 |
|------|------|
| 电影级摄影机 | 电影质感，暗部层次丰富 |
| 高端数码相机 | 高清广告感 |
| 徕卡色调 | 温润复古 |
| 蔡司镜头 | 锐利通透 |

### B. 运镜必须用中文Seedance指令公式
```
❌ 近景平视微手持
✅ 近景平视，镜头缓缓推近，手持拍摄镜头轻微晃动
```

**Seedance 2.0 中文运镜速查**：

| 需求 | 提示词 |
|------|--------|
| 聚焦紧张 | 镜头缓缓推近 |
| 揭示孤独 | 镜头缓缓拉远 |
| 仪式登场 | 镜头环绕主体旋转 |
| 跟拍纪实 | 手持拍摄，镜头轻微晃动 |
| 沉浸特写 | 长焦镜头，背景虚化突出主体 |

**万能运镜公式**：`镜头类型 + 运动方向 + 速度 + 伴随效果`
示例：镜头缓缓推近，聚焦到人物面部，伴随轻微手持晃动

### C. 三层构图法（每条提示词必用）
```
前景虚化[物体]轻微遮挡画面[方位]，中景[主体]清晰对焦，背景[环境]柔化模糊
```
- **前景遮挡物本身就是叙事**：绿植=生活气息，栏杆=困境，消防栓=街头，纱帘=偷窥，咖啡杯=社交

### D. 感官细节（写不满时优先加这个）
每条至少加 1 个**听觉/触觉/动态**感官：
- 听觉：远处隐约的车流声、键盘敲击声、鸟鸣声
- 触觉：微风拂过发丝、阳光晒在后颈的温热
- 动态：咖啡热气缓缓升起、裙摆随走路轻轻摆动

### E. 缺陷美学（真实=不完美）
以下元素至少选 2 个加入画面：
- 镜头轻微呼吸感晃动
- 画面边缘有极微暗角
- 阳光下空气中有细微灰尘颗粒
- 背景光斑自然散开
- 运动模糊（快速移动时）
- 轻微色散（强光源边缘）
- 焦点微微调整过程

### F. 万能写实公式（开场一句话模板）
```
[镜头参数]+[拍摄设备]质感，[运镜方式]。
[时间段]，[光影方向+质感]。
[皮肤肌理词]，[景深虚实词]，[画面瑕疵词]。
```
示例：
> 50mm镜头，电影级摄影机质感，手持拍摄镜头轻微晃动。
> 黄金时刻暖调，日落前柔和侧逆光勾勒人物轮廓。
> 原生皮肤纹理可见，毛孔细节真实，无过度磨皮。
> 前景虚化枝叶轻微遮挡，主体清晰对焦，背景自然柔化。
> 画面微带胶片颗粒感。

## 执行流程

收到 `主题 + CEFR 等级` 时：

1. **构思**：2 秒想清楚谁在哪干什么、核心句塞哪里
2. **写台词**：4 句英文，核对 CEFR，估算时长（英语 ≈ 2.5-3 秒/句）
3. **分镜**：3-4 个镜头，分配台词
4. **写动作链**：每个镜头的情绪动作链（眼神→面部→身体→收尾）
5. **选光效**：根据场景选 5 种光效之一
6. **写三层构图**：至少一个镜头有前景遮挡+中景对焦+背景虚化
7. **加物理细节**：惯性/重力/风阻/声音暗示
8. **加缺陷美学**：镜头晃动/灰尘/运动模糊
9. **写单段中文 prompt**：按模板整合，1800-2000 字符
10. **自检清单**
