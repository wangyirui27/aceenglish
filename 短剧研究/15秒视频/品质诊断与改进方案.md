# 品质诊断：我们的提示词哪里有"AI 味"

## 当前问题（以 S124 为例）

### ❶ 开头套公式，不是在描述场景
```
❌ 日光电影卷日光电影卷风格，24帧，35mm胶片细腻颗粒感，
   PBR皮肤材质毛孔级可辨、有自然血色和微油光、绝无塑料感磨皮，
   摄影写实主义，非CG，非动漫，非风格化。
```
→ 这是"模板标签堆叠"，所有条目的提示词都长得一样，平台算法一看就知道是流水线 AI 生成。

### ❷ 光源描述只写了"参数"，没写"效果"
```
❌ 色温约5600K，顶部航站楼荧光灯补光色温略冷（4000K）形成微妙冷暖混合
```
→ 方法论说了：光源描述公式 = **光从哪来 → 照到什么 → 产生什么效果**。
→ 我们只写了"5600K"，没写"在 Leo 颧骨上形成一个柔和的高光三角区，下巴处留有干净的阴影"。

### ❸ 材质描述用了"万能公式"，每个角色都一样
```
❌ Leo: PBR皮肤材质毛孔级可辨、有自然血色和微油光
❌ Sarah: PBR皮肤材质毛孔级可辨、有自然血色和微油光
```
→ 两个人的皮肤材质描述完全一样！不同角色应该有不同的皮肤状态。
→ Leo 赶路疲惫 → 鼻翼两侧毛孔扩张有油光，眼角微红充血
→ Sarah 制服职业 → 皮肤状态良好，妆容自然，口红微微晕染

### ❹ "摄影写实，非CG，非动漫"—— 这是 2024 年的写法
```
❌ 摄影写实，非CG，非动漫，非风格化
```
→ 方法论的核心：**约束 > 描述**，但更好的方式是把约束融进具体描述里。
→ 写"皮肤上清晰可见粉底遮盖后仍透出的毛孔纹理"比写"非CG，非动漫"有效 10 倍。

### ❺ 每个镜头都重复写光源（碎片化）
```
❌ 镜头1: 玻璃窗侧光照亮他右脸形成柔和高光
❌ 镜头2: 顶部荧光灯在她前额形成柔和高光
❌ 镜头3: 侧光在他面部轮廓形成一圈暖色逆光勾边
❌ 镜头4: 玻璃窗自然光均匀照亮她整张脸
```
→ 光源应该在**场景描述段统一写清楚**，每个镜头只需要写"因光线导致的具体效果"。

### ❻ 环境细节"堆砌"而非"控制"
```
❌ 背景右侧落地玻璃窗外停机坪停靠一架白色客机和地勤车辆柔和虚化
❌ 左侧蓝色PU候机座椅几位旅客背影动态模糊
❌ 远处登机口闸机和航站楼指示牌隐约可见
❌ 米白磨砂地砖反射荧光灯光
```
→ 方法论：**每个元素必须有存在的物理或叙事理由**。
→ 窗户 → 不是"有一扇窗"，而是"窗户的自然光在 Leo 脸上产生明暗分割"。
→ 指示牌 → 不是"看得见"，而是"远处 LED 指示牌散发冰冷蓝光，暗示他还没到目的地"。

### ❼ 否定词冗长
```
❌ 禁止字幕、文字叠加、水印、LOGO、航司名、音乐、卡通、动漫、
   塑料感皮肤、过度磨皮、夜晚、暴力、武器、血腥、脏乱、破损设施
```
→ 17 个否定词，一半跟场景无关（夜晚？暴力？血腥？机场候机厅为什么要排除这些？）
→ 只排除该场景可能误出现的东西。

## 改进策略

### A. 开头重构：从"标签堆叠"变成"场景定义"
```
❌ 旧: 日光电影卷日光电影卷风格，24帧，35mm胶片细腻颗粒感，PBR皮肤...

✅ 新: 
场景摄影美学：纪录片式自然摄影，用日光电影卷日光电影卷拍摄，
自然颗粒感而非数码平滑，画面呈现美国西海岸午后特有的暖调自然光
与航站楼人工冷光源的微妙混合。
```
→ "纪录片式自然摄影"比"摄影写实主义"有调性
→ "美国西海岸午后暖调"比"色温5600K"更有画面感

### B. 光源描述：单独一段，写清物理因果
```
✅ 光线场景：
左侧大面积落地玻璃窗引入午后自然侧光（约5600K），在人物面部
形成清晰的明暗分割；头顶航站楼荧光灯（约4000K）从上方补光
在额头和鼻梁形成冷色高光，与窗光形成冷暖混合；
远处停机坪的地面反光在窗外制造微弱的蓝白色环境反射。
```

### C. 材质：按角色差异化
```
✅ Leo 的皮肤：赶路后额头毛孔扩张，鼻翼两侧油光可见，
   下巴胡茬根部青色清晰，眼角微红充血显示疲劳。

✅ Sarah 的皮肤：妆容整洁但非浓妆，粉底轻薄遮盖后
   仍可见鼻翼毛孔，口红在嘴角有极微晕染，
   制服面料在肩部有自然坐姿褶皱。
```

### D. "不完美细节"：给每个不完美一个叙事理由
```
Leo 的西装领口敞开 → 因为赶路所以敞开透气
Sarah 制服口袋里充电线 → 她随身带充电线是职业习惯
咖啡杯有磨损 → 不是新杯子，是常用
```

### E. 每个镜头只写因光线导致的效果，不重复写光源来源
```
✅ 镜头1: 午后窗光在他右颧骨形成一小块柔和高光三角区，
         左侧陷入温暖阴影，手机红色微光映在下巴（不再重复写"玻璃窗侧光"）
```

### F. 环境细节：每个元素写"存在的理由"
```
✅ 落地窗（光源来源）→ 午后自然光的主要来源，照射 Leo 面部
✅ 远处停机坪飞机（叙事意义）→ 暗示旅程未完，他需要手机联系
✅ 荧光灯（光效来源）→ 在 Sarah 制服上形成冷色反光
```

### G. 否定词：只排除该场景可能误出现的
```
❌ 旧: 禁止字幕、文字叠加、水印、LOGO、航司名、音乐、卡通、动漫、
       塑料感皮肤、过度磨皮、夜晚、暴力、武器、血腥、脏乱、破损设施

✅ 新: 无字幕无水印无配乐，禁止卡通动漫塑料感过度磨皮，
       禁止画面出现航司真实品牌标识。
```
→ 17 个字 → 12 个字，砍掉无关的
