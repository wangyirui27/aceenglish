## 4 套公式

### 公式 1：基础万能公式（新手直接用，适合文生视频）

```
主体 + 场景 + 动作 + 风格 + 光线 + 运镜 + 画质 + 比例 + 细节
```

**示例：**
> 白衣侠客在雨夜古街持剑独行，武侠电影质感，冷色调侧逆光，镜头缓慢推进跟拍，4K超清，电影级景深，竖屏9:16，流畅自然

---

### 公式 2：电影级运镜公式（拉满大片感）

```
景别 + 运镜 + 主体 + 场景 + 光影 + 风格 + 画质 + 参数
```

**示例：**
> 全景→慢推→特写，环绕运镜，赛博朋克少女在霓虹天台跳舞，蓝紫霓虹光+丁达尔光束，王家卫风格，8K，60帧，16:9横屏

**关键创新点：** 景别变化写成"全景→慢推→特写"链式，让 AI 理解镜头过渡。

---

### 公式 3：分镜连贯公式（解决画面跳变）

```
首帧定帧 + 转场描述 + 下镜提示 + 一致性约束
```

**示例：**
> 首帧：黄昏海边少女奔跑，暖金色逆光，全景慢拉；转场：淡入淡出1秒；下镜：特写海浪拍打脚踝，慢动作，保持人物服装/发色一致，4K，竖屏9:16

**核心要点：** 最后一句**"保持一致"**别漏，专治多镜头拼接的穿帮。

---

### 公式 4：图生视频公式（人物场景最稳）

```
参考图描述 + 动作延续 + 运镜 + 风格 + 画质 + 一致性
```

**示例：**
> 以这张古风美人图为首帧，人物缓步走向桃花林，镜头环绕跟拍，古风仙侠，柔光，4K，保持面部/服饰一致，流畅运镜

**核心要点：** 用参考图做首帧，角色崩坏概率显著降低。

## 完整操作流程（4 步）

### 第 1 步：豆包写分镜脚本

**指令模板（直接复制用，替换主题/时长/比例即可）：**

```
你是专业电影导演，帮我写一个【30秒、9:16竖屏】短视频分镜，共6个镜头，
每镜标注：时长、景别、运镜、画面描述（含主体、动作、场景、光线、风格、画质）、
音效、旁白；要求运镜专业，画面连贯，电影级质感
```

**输出示例：**
> 0-5s，全景，缓慢推镜：赛博朋克少年站在雨夜霓虹街头，蓝紫霓虹光，冷色调，4K……

### 第 2 步：即梦文生图生成首帧

- 设好比例和分辨率
- 每个镜头生成一张静态图，挑最满意的保存作首帧

### 第 3 步：即梦图生视频（核心）

- 推荐**图生视频模式**，上传首帧 + 输入运镜和动作提示词
- **运动幅度建议选低或中**（幅度太大容易崩）
- 每段 5-10 秒，多镜头依次生成
- 无参考图也可直接**文生视频**，分段粘贴豆包的分镜提示词

### 第 4 步：剪映后期

- 导入所有片段
- 加转场、字幕、BGM
- 调色导出
