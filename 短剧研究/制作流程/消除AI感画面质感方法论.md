## 核心观点

AI 绘图的特质就是"平"——纯靠代码想象，光线平、焦点平、各方面都喜欢但就是一股 AI 味。

**真实 = 不完美**：皮肤的暗斑/雀斑、材质的凹坑/油斑/锈迹、氛围的斑驳光线/空气尘埃。这些不完美才是真实的来源。

---

## 三层方法论

### 第一层：真实感 = 物理语言，不是风格语言

AI 不懂"电影感"，但它懂**物理规律**。

**核心**：描述物理现象，而不是描述你想要的感觉。

| ❌ 风格语言（AI 听不懂） | ✅ 物理语言（AI 能理解） |
|--------------------------|--------------------------|
| 电影感 | 35mm 胶片拍摄，颗粒感，浅景深 |
| 复古色调 | 钨丝灯色温 3200K，暖光偏黄 |
| 高级感 | PBR 材质，皮下散射，光线折射 |

### 第二层：氛围感 = 控制，不是堆砌

氛围感不是加得越多越好，而是**越有控制越高级**。

**原则**：每个元素必须有存在的物理或叙事理由：
- 烟雾在这里 → 因为厨房火一小时前熄灭
- 这里是暗的 → 因为没有光源能照到这里
- 水坑是浑浊的 → 因为雨后有悬浮沉积物

**没有理由的效果，一律不写。**

### 第三层：约束 > 描述

告诉 AI "不要做什么"和"只能在哪里做"，比告诉它"要做什么"更有效。

| ❌ 描述性指令 | ✅ 约束性指令 |
|--------------|--------------|
| add some rust | rust ONLY at cut edges and fastener heads |
| dramatic atmosphere | NOT dramatic. NOT heroic. NOT cinematic-action |
| empty background | NO human figures. NO people. NO humanoid shapes anywhere |

**核心**：用约束锁死 AI 的自由发挥空间。

## PBR 材质描述模板

### 布料

```
PBR材质——布料：身着层叠的棉质长袍，色调沉稳，呈暗海军蓝与灰色交织；
布料的垂坠感与重量感清晰可见——体现在肩部的自然垂落以及手肘处的褶皱堆叠；
在烛光的映照下，粗纺布料的纹理质感纤毫毕现；
衣物显露长途跋涉后的风尘痕迹，细微的尘埃积淀于衣褶深处。
```

### 皮肤

```
PBR材质——皮肤：耳廓软骨及下颌边缘处可见逼真的皮下散射（Subsurface Scattering）效果；
皮肤质感饱经风霜，毛孔级别的微细纹理清晰可辨；
高光反射变化自然——额头部位略显油光，绝无那种"塑料感"般的过度磨皮修饰；
下巴处隐约可见短须的青色胡茬。
```

### 金属/皮质

```
PBR材质——金属：剑身收纳于磨损斑驳的皮质剑鞘之中；
皮质剑带缝线清晰可见，呈现出经年使用的自然包浆感，绝无任何人工做作的光泽。
```

## 光源描述公式

必须写清楚**光从哪来、照到什么、产生什么效果**：

```
从窗户/门口射入的月光/阳光——造成明暗对比
桌上的烛光/油灯——暖色点光源，照亮人物面部
烟雾/尘埃——被光线照亮形成丁达尔效应
```

**关键**：每个光源都要有物理来源，不能凭空出现"戏剧性光线"。

## 提示词编写 Checklist

- [ ] 用物理语言替代风格语言
- [ ] 每个光源都有物理来源
- [ ] 每个氛围元素都有叙事理由
- [ ] 加入 PBR 材质描述（布料/皮肤/金属）
- [ ] 加入不完美细节（尘埃/磨损/锈迹/油光）
- [ ] 用约束性指令限制 AI 自由发挥
- [ ] 排除不需要的风格标签（非游戏、非CGI、非风格化）
