## 模块 1：主体（定人、定物、定形象）

白裙赤脚的女孩 / 披风猎猎的武士 / 猫耳二次元少女

## 模块 4：运镜（定景别、定机位、定运动）

**核心：** Seedance 2.0 对运镜指令识别准确，直接写出运镜方式即可获得相应效果。

### 景别选择
```
大远景 / 全景 / 中景 / 近景 / 特写 / 大特写 /
中近景（膝上）/ 人物全景
```

### 基础运镜动作词

| 运镜 | 提示词 |
|------|--------|
| **推** | 镜头缓缓推进 / 平滑推近至特写 / 焦点逐渐锁定主体 |
| **拉** | 镜头向后平稳拉远 / 缓慢拉远展现场景全貌 / 史诗式后拉 |
| **移** | 向左/右平稳横移 / 跟拍平移 / 水平匀速移动 |
| **跟** | 稳定跟拍 / 主体居中追踪 / 背景运动模糊 |
| **摇** | 镜头缓慢上摇/下摇 / 水平横摇扫视 |
| **升降** | 镜头垂直抬升 / 低角度升起至高空俯瞰 |
| **环绕** | 环绕主体一周 / 半环绕运动 / 低机位仰角环绕 |

### 复合与进阶运镜

**组合运镜：**
```
缓慢推进的同时轻微环绕 / 后拉过程中镜头微微抬升
```

**风格运镜：**
```
斯坦尼康级丝滑稳定 / 手持微震动的呼吸感 / 盗梦空间式轴向旋转
```

**视角描述：**
```
第一人称POV主观视角 / 无人机鸟瞰 / 低角度仰拍 / 过肩跟随视角
```

### 运镜稳定约束（必加）
```
固定镜头 / 画面流畅不抖动 / 丝滑无卡顿 / 运镜平稳顺滑
```

## 使用工作流（每次生成前 1 分钟）

1. 先确定**主体是谁**
2. 写清楚**动作怎么动**（记得用慢的、连贯的描述）
3. 交代**场景在哪里**
4. 加上**运镜怎么动**
5. 最后补上**风格和画质词**收尾

## 核心记忆点

- **结构公式**：主体 + 动作 + 场景 + 运镜 + 风格
- **动作铁律**：宁慢勿快，宁简勿繁（缓慢、轻柔、连贯、自然）
- **主体防变形**：必加约束词（五官比例正常/面部稳定不扭曲/服装一致）
- **运镜约束**：必加稳定词（画面流畅不抖动/丝滑无卡顿）
