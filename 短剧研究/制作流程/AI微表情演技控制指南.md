## 核心观点

AI 角色表情经常崩坏——要么过于夸张，要么呆若木鸡。解决方案：**不要直接写情绪词（开心/伤心），要把情绪翻译成具体的面部肌肉运动指令**。

---

## 三步法

### 第一步：描述面部肌肉运动

人类的表情本质是面部肌肉的运动。不同肌肉组合呈现不同情绪。

**关键参数**：
- 嘴角上扬的幅度
- 眨眼的频率
- 是否皱眉、皱眉的程度
- 头部姿态
- 辅助性手部动作

### 第二步：为情绪配上肢体语言

每种情绪都有自然的肢体反应：
- 害羞 → 下意识捂嘴
- 犹豫 → 玩手指、轻咬嘴唇
- 难过 → 吸鼻子、低头
- 坚定 → 下颌收紧、肩膀稳定

### 第三步：微表情是变化的过程

情绪不是一蹴而就的，而是逐步发展的。以"害羞的笑"为例：

> 平静自然 → 被夸奖 → 嘴角眼角微微上扬 → 弧度变大 → 捂嘴 → 视线低头

**层层递进才是"演技"的精髓。**

### 特别提示

每个角色性格不同，面对同一情绪的回应方式也不同。要根据角色设定量身定制面部肌肉、肢体语言与情绪节奏。

## 8 种影帝级微表情提示词

### 1. 开心（内敛、不夸张）

```
年轻东亚女性，轻微开心，嘴角微微上扬，眼角出现细微鱼尾纹，
眼轮匝肌自然收缩，眼神明亮，轻微眨眼，肩膀轻微起伏带呼吸感，
5 秒情绪缓慢舒展，电影布光，4K 高清，皮肤纹理真实。
```

### 2. 难过（隐忍不哭）

```
30 岁东亚男性，眼眶微红，泪水在眼眶打转但强忍不落，
下唇轻微颤抖，咬住嘴唇，低头，肩头微微耸动，缓慢呼吸，
喉结轻微滚动，5 秒情绪克制积累，电影质感，4K。
```

### 3. 疑惑（思考不解）

```
青年女性，疑惑思考，眉头微蹙，眼球轻微转动，
眼神聚焦又散开，嘴唇轻抿，头微侧，缓慢呼吸，
短暂停顿思考，5 秒疑惑递进，柔和光影，4K。
```

### 4. 害怕（恐惧不安）

```
年轻女性，害怕不安，眼神轻微飘移，瞳孔微缩，
呼吸变浅急促，嘴唇微颤，吞咽动作，额头细微出汗，
身体轻微僵硬，5 秒恐惧递进，弱光氛围，4K。
```

### 5. 惊吓（突然受惊）

```
东亚少年，突然惊吓，0-1 秒表情凝固僵住，
1-2 秒眼睛猛然睁大瞳孔扩张，2-3 秒眉毛上扬，
3-4 秒呼吸骤停后急促，4-5 秒身体微缩，
5 秒瞬间受惊反应，真实微表情，4K。
```

### 6. 惊喜（由惊到喜）

```
年轻女性，惊喜情绪演变，0-1 秒愣住瞳孔放大，
1-2 秒嘴角快速上扬，2-3 秒眼睛弯起带泪光，
3-4 秒手轻捂嘴，4-5 秒放松微笑，
5 秒惊喜递进，暖光，4K。
```

### 7. 厌恶（轻微嫌弃）

```
成年男性，轻微厌恶，鼻翼微收，上唇轻微抬起，
眼神回避，眉头微蹙，头轻微后仰，呼吸轻微停顿，
无夸张表情，5 秒内敛厌恶，真实皮肤，4K。
```

### 8. 坚定（眼神有力量）

```
东亚女性，坚定沉稳，眼神直视焦点不闪躲，下颌收紧，
嘴唇紧闭，呼吸平稳，肩膀稳定，眉毛细微收紧显决心，
5 秒坚定气场，正面硬光，4K。
```

## 提示词编写公式

```
[人物身份]，[情绪名称]，[面部肌肉细节1]，[面部肌肉细节2]，
[肢体语言1]，[肢体语言2]，[呼吸/生理反应]，
[时间长度] [情绪变化节奏]，[光影氛围]，[画质参数]。
```

**关键原则**：
1. 写肌肉运动，不写情绪词
2. 加入时间线递进（0-1秒...1-2秒...）
3. 包含呼吸和微生理反应（吞咽、瞳孔、出汗）
4. 指定光影和画质
