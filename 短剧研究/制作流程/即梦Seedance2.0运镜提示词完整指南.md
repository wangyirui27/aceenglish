## 核心观点

同一个 Seedance 2.0 模型，别人出大片你出废片——核心差距在**提示词缺少"导演级"运镜逻辑**。Seedance 2.0 对运镜指令识别率非常高，只要用对提示词就能精准生成高质量视频。

**运镜是 AI 视频的灵魂。** 好视频的核心不在于画面有多好看，而在于镜头如何运动、如何切换、如何组织。

## 一、基础运镜提示词（8种）

| 运镜类型 | 提示词指令 | 效果说明 |
|----------|-----------|----------|
| **推镜头** | 镜头缓缓推近 | 逐渐聚焦，制造紧张感/关注感 |
| **拉镜头** | 镜头缓缓拉远 | 逐渐远离，揭示全貌/孤独感 |
| **上摇** | 镜头缓缓上摇 | 视线上移，展现高度/仰望感 |
| **下摇** | 镜头缓缓下摇 | 视线下移，俯瞰/压迫感 |
| **左横移** | 镜头向左缓缓横移 | 水平扫视，展现空间宽度 |
| **右横移** | 镜头向右缓缓横移 | 水平扫视，展现空间宽度 |
| **环绕** | 镜头环绕主体旋转 | 360度展示，仪式感/震撼感 |
| **跟随** | 镜头跟随主体移动 | 跟拍，动态叙事/追逐感 |

## 四、组合运镜提示词

将多种运镜组合使用，创造更丰富的镜头语言。

### 常用组合模式

| 组合 | 提示词 | 效果 |
|------|--------|------|
| 推+环绕 | 镜头缓缓推近并环绕主体 | 聚焦中的仪式感 |
| 拉+上摇 | 镜头缓缓拉远并上摇 | 从细节到宏大全景 |
| 跟随+推 | 镜头跟随主体并缓缓推近 | 动态追逐中的聚焦 |
| 环绕+升降 | 镜头环绕并从低处升起 | 环绕上升的史诗感 |
| 手持+跟随 | 手持跟拍，镜头跟随主体 | 纪实追逐，紧张真实 |
| 航拍+下降 | 航拍视角从高空缓缓下降 | 从全局到聚焦的揭示 |

### 组合运镜提示词模板

```
[基础运镜1] + [基础运镜2] + [修饰词] + [速度]
```

示例：
> 镜头环绕主体旋转并缓缓推近，速度逐渐加快，最后定格在面部特写

## 五、万能运镜提示词公式

### 公式结构

```
镜头类型 + 运动方向 + 速度 + 伴随效果
```

### 公式示例

| 维度 | 选项示例 |
|------|----------|
| 镜头类型 | 推/拉/摇/移/环绕/跟随/升降/手持/航拍/FPV |
| 运动方向 | 向前/向后/向上/向下/向左/向右/环绕/螺旋 |
| 速度 | 缓缓/快速/逐渐加速/逐渐减速/匀速 |
| 伴随效果 | 聚焦到面部/揭示全景/背景虚化/光影变化 |

### 万能模板

> [镜头类型] + [运动方向] + [速度]，[伴随效果]

**示例：**
> 镜头从低处缓缓升起并环绕主体旋转，速度逐渐加快，最终定格在俯瞰全景
