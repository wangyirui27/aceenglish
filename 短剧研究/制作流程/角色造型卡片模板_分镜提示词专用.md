# 角色造型卡片模板 — 分镜提示词专用

> 用途：每次写分镜提示词时，直接复制对应角色的「外在描述段」到提示词中，确保跨镜头造型一致性。
> 来源：基于《AI角色生成中服装道具设计与整体造型提示词体系全解析》的四层结构公式。

---

## 使用方法

1. 为每个主要角色填写一张「造型卡片」
2. 出分镜提示词时，把卡片中的【外在描述段】整段粘贴进提示词
3. 每条分镜只改「姿势/表情/动作/场景/镜头」，造型描述保持不变
4. 如需换装（剧情需要），在原卡片基础上另建「换装变体」

## 快速填写检查清单

填写角色造型卡时，逐项确认：

- [ ] **年龄+体型** — 写了吗？（影响AI对比例的判断）
- [ ] **发型锚点** — 写到具体颜色和长度了吗？（跨镜头最易识别的特征）
- [ ] **服装廓形** — 写了吗？（A字摆？收腰？oversize？）
- [ ] **服装材质** — 写了吗？（丝绸？亚麻？皮革？直接影响质感）
- [ ] **颜色** — 写了吗？（别只写"蓝色"，写"藏青"或"粉蓝"更精准）
- [ ] **关键装饰/纹样** — 写了吗？（有就写，没有就不写，别硬凑）
- [ ] **配饰** — 写了吗？（眼镜、手表、耳饰、武器等）
- [ ] **画风锚点词** — 写了吗？（赛璐璐/胶片/水墨/3D黏土 等）

## 跨镜头使用规则

| 场景类型 | 外在描述段 | 改动部分 |
|---------|-----------|---------|
| 同场景不同镜头 | ✅ 完全复制 | 只改镜头角度（特写/中景/远景） |
| 换场景 | ✅ 完全复制 | 只改场景背景描述 |
| 剧情换装 | ⚠️ 复制后修改服装部分 | 另建换装变体卡 |
| 时间跳跃（闪回等） | ✅ 复制 | 加对应年龄/状态修饰词 |

## 多角色短剧的工作流

1. **先建所有角色的造型卡**（按上面模板填写）
2. **写分镜脚本时**，每条分镜标注角色名，出提示词时替换为对应造型卡
3. **有对话/互动的分镜**，把两个角色的外在描述段用逗号连接，再加互动描述
4. **存档**：所有造型卡统一放在本文件中，后续项目直接复制修改
