## 核心背景

- 即梦 seedance2.0「全能参考」模式生成真人视频时，容易触发审核
- AI 仿真人漫剧在**红果平台**的分成系数最高（60），所以真人风格是高收益方向
- 本文总结 9 种实测过审的方法

---

## 思路一：角色多视图设定卡（信息轰炸）

**原理**：用大量角色参考图"轰炸"模型，让它理解角色身份，同时用设定卡形式降低审核触发概率。

### 方法 ① 角色面部特写 + 全身三视图 + 6 肖像 21:9（推荐首选）

```
21:9超宽横版角色完整设定卡，纯白干净棚拍背景。以参考图人物为唯一身份锚点：
脸型轮廓（下颌线、颧骨、下巴形状）、眼型、眉形、鼻梁与鼻翼、嘴唇厚薄与嘴角形状、
年龄气质必须严格一致；发际线、发型结构与发饰必须严格一致。
只允许同一个角色，禁止换脸、禁止五官漂移、禁止发型简化或发饰缺失。
```

**个人经验**：正面没问题但低头容易幻视成其他人，所以增加低头/抬头角度锁定角色。

### 方法 ② 全身视图 + 6 肖像 16:9

```
16：9横版角色完整设定卡，纯白干净棚拍背景。超模身材，身材修长。
以参考图人物为唯一身份锚点：脸型轮廓（下颌线、颧骨、下巴形状）、
眼型、眉形、鼻梁与鼻翼、嘴唇厚薄与嘴角形状、年龄气质必须严格一致；
发际线、发型结构与发饰必须严格一致。
只允许同一个角色，禁止换脸、禁止五官漂移、禁止发型简化或发饰缺失。
```

**适用场景**：方法 ① 的 21:9 面部特写不通过时，换这个。


## 思路二：风格转绘、人脸和服装分离

**原理**：用非真人风格（素描、彩铅）展示面部，用白模人台展示服装，规避人脸识别审核。

### 方法 ⑤ 素描线稿头部 + 无脸白模人台三视图

```
专业角色设计规格说明图（Character Design Spec Sheet），
```

左栏头部用素描线稿，右栏三视图用无脸白模人台，通过色板和局部特写传递角色信息。

### 方法 ⑥ 彩铅风格肖像 + 面部移除的三视图

```
专业角色设定参考图，16:9横版构图，浅灰色干净棚拍背景，背景带细腻等距网格线，
```

左栏头部用彩铅呈现，右侧仅面部区域白模化，其余全身维持真实摄影质感。

## 思路三：提示词内容策略

### 方法 ⑦ 文案绕过限制

**关键**：首先告诉即梦你是用 AI 生成的人物（声明创作意图）。

**文案避坑**：
- ❌ 不要用「A图片人物在B图片环境，模仿C图片人物做动作」——容易触发违规
- ✅ 应该给真实语境，比如：「现代霸道总裁实拍短剧，9:16，电影级写实质感」

### 方法 ⑧ 分段生成

分段生成单镜头而非一次性生成完整场景，降低审核敏感度。

### 方法 ⑨ 图片预处理

对参考图片进行适当处理（如调色、裁剪），降低被识别为真人照片的概率。
