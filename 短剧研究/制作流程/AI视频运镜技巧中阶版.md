## 核心价值

运镜是提升视频电影感和专业度的关键技巧。中阶运镜在基础推拉摇移基础上，通过**组合运镜**和**特殊视角**创造更强的视觉冲击力。

---

## 5 种中阶运镜技巧

### 1. 盗梦空间 + 旋转

**效果：** 强烈的视觉冲突感，适合建筑、空间、超现实场景

**提示词：**
> 镜头旋转、拉近，盗梦空间运镜，强烈的视觉冲突

**适用场景：**
- 建筑外观展示
- 空间转换
- 超现实/梦境场景
- 片头/转场

**案例描述：**
从建筑底部开始，以螺旋上升的方式拍摄，完整呈现建筑外观的细节、线条结构，展示建筑与周边环境的空间关系。

---

### 2. 推 + 拉 + 环绕（组合运镜）

**效果：** 多角度展示，节奏感强，适合动作/科幻场景

**提示词：**
> 镜头推进，聚焦到人物面部，从斜侧角度拉运镜头，环绕拍摄飞船尾部

**适用场景：**
- 科幻/动作场景
- 角色登场
- 载具/机械展示
- 战斗/追逐场面

**案例描述：**
科幻电影，曹操和自己的大将们乘着高科技飞行器极速飞行，无边无际的军队乘着飞行器跟着后面，有很多战旗写着曹。

---

### 3. 俯拍 + 跟随镜头

**效果：** 鸟瞰视角 + 动态跟随，适合飞行、高速移动场景

**提示词：**
> 俯拍镜头，镜头缓慢推进，镜头跟随人物继续朝向前方飞行

**适用场景：**
- 飞行场景
- 高速移动
- 追逐场面
- 展示环境规模

**案例描述：**
未来世界，一位智者在巨型机械蜻蜓上疾速飞行。骑手俯身紧贴蜻蜓背部，以近乎垂直的俯冲姿态穿行。

---

### 4. FPV 运镜

**效果：** 第一人称视角沉浸感，适合追逐、战斗、高速场景

**提示词：**
> FPV运镜方式，人物骑着马继续向前方奔跑

**适用场景：**
- 追逐场面
- 战斗场景
- 第一人称视角
- 速度感展示

**案例描述：**
先锋艺术，视觉设计，时光隧道，旋转粒子，千万铁骑大军冲锋陷阵场面，轻盈空间、画面平铺，视觉动向、Liuhua风格，平面流动、震撼场面，高级唯美。

---

### 5. 螺旋下降镜头

**效果：** 深邃、神秘、压迫感，适合悬疑/恐怖/探索场景

**提示词：**
> 镜头快速推进到人物后，镜头以螺旋下降的方式从人物向下拍摄

**适用场景：**
- 悬疑/恐怖场景
- 深度/高度展示
- 神秘探索
- 心理压迫感

**案例描述：**
顶视图，在迷雾中隐约悬浮着一座无限循环的悬梯，一位神秘小人行走在悬梯上，悬梯深不见底，下方被浓密的雾气掩埋，整个画面透着灰蓝色阴暗气息，厚厚的灰尘，古老斑驳的质感。

## 运镜组合原则

### 组合逻辑

| 组合类型 | 效果 | 适用场景 |
|----------|------|----------|
| 推 + 旋转 | 聚焦 + 视觉冲击 | 角色登场、关键道具 |
| 拉 + 环绕 | 环境揭示 + 多角度展示 | 场景介绍、空间感 |
| 俯拍 + 跟随 | 鸟瞰 + 动态感 | 追逐、飞行、高速移动 |
| FPV + 推进 | 沉浸 + 紧迫感 | 战斗、追逐、紧张时刻 |
| 螺旋 + 下降 | 深度 + 神秘感 | 悬疑、探索、心理压迫 |

### 运镜节奏

- **慢速运镜：** 适合情绪铺垫、环境展示、浪漫/温柔场景
- **中速运镜：** 适合叙事推进、角色互动、日常场景
- **快速运镜：** 适合动作场面、紧张时刻、视觉冲击

## 提示词编写技巧

### 运镜提示词结构

```
[镜头运动方式] + [运动方向] + [速度] + [伴随效果]
```

### 示例

**基础版：**
> 镜头推进

**中阶版：**
> 镜头缓慢推进，聚焦到人物面部，伴随轻微旋转

**高阶版：**
> 镜头从全景缓慢推进，聚焦到人物面部特写，伴随螺旋上升运动，背景逐渐虚化，光影变化强调面部轮廓

### 常用修饰词

| 修饰词 | 效果 |
|--------|------|
| 缓慢/slowly | 情绪铺垫、优雅感 |
| 快速/rapidly | 紧张感、冲击力 |
| 平稳/smoothly | 专业感、电影感 |
| 手持/handheld | 纪实感、紧张感 |
| 环绕/orbiting | 多角度展示、仪式感 |
| 螺旋/spiral | 超现实、视觉冲击 |
| 俯冲/diving | 速度感、冲击力 |
