公式一：基础万能公式（新手直接用）

适合文生视频，结构很简单：

主体 + 场景 + 动作 + 风格 + 光线 + 运镜 + 画质 + 比例 + 细节

举个例子——

"白衣侠客在雨夜古街持剑独行，武侠电影质感，冷色调侧逆光，镜头缓慢推进跟拍，4K超清，电影级景深，竖屏9:16，流畅自然"

把主体和场景换成你想要的内容，直接套就行。

公式二：电影级运镜公式（拉满大片感）

想让画面更有冲击力？加上景别变化和运镜指令：

景别 + 运镜 + 主体 + 场景 + 光影 + 风格 + 画质 + 参数

比如："全景→慢推→特写，环绕运镜，赛博朋克少女在霓虹天台跳舞，蓝紫霓虹光+丁达尔光束，王家卫风格，8K，60帧，16:9横屏"

这套公式出来的画面，质感直接拉满。

公式三：分镜连贯公式（解决画面跳变）

多镜头拼接最怕前后不一致，这套公式专治这个问题：

首帧定帧 + 转场描述 + 下镜提示 + 一致性约束

示例："首帧：黄昏海边少女奔跑，暖金色逆光，全景慢拉；转场：淡入淡出1秒；下镜：特写海浪拍打脚踝，慢动作，保持人物服装/发色一致，4K，竖屏9:16"

关键就在最后那句"保持一致"，别漏了。

公式四：图生视频公式（人物场景最稳）

手里有参考图的话，这套最靠谱：

参考图描述 + 动作延续 + 运镜 + 风格 + 画质 + 一致性

比如："以这张古风美人图为首帧，人物缓步走向桃花林，镜头环绕跟拍，古风仙侠，柔光，4K，保持面部/服饰一致，流畅运镜"

用参考图做首帧，角色崩坏的概率会小很多。

      
    

四步出片，完整操作流程

第一步，也是最关键的一步——让豆包帮你写分镜脚本。

直接把下面这段指令复制给豆包，改掉主题、时长和比例就行：

"你是专业电影导演，帮我写一个【30秒、9:16竖屏】短视频分镜，共6个镜头，每镜标注：时长、景别、运镜、画面描述（含主体、动作、场景、光线、风格、画质）、音效、旁白；要求运镜专业，画面连贯，电影级质感"

豆包会输出类似这样的内容：

0-5s，全景，缓慢推镜：赛博朋克少年站在雨夜霓虹街头，蓝紫霓虹光，冷色调，4K……

这些提示词可以直接粘贴到即梦里用。

第二步，用即梦的"文生图"功能，把每个镜头的提示词分别生成静态图，作为首帧。设好比例和分辨率，挑满意的保存。

第三步，核心生成环节。推荐用"图生视频"模式，上传刚才的首帧图，再输入运镜和动作提示词。运动幅度建议选低或中，幅度太大容易崩。每段5-10秒，多镜头依次生成。

没有参考图的话，也可以直接用"文生视频"，把豆包写的分镜提示词分段粘贴进去。

第四步，打开剪映，导入所有视频片段。加转场、字幕、BGM，调个色，导出成片。

整个流程就这么简单……真的。
