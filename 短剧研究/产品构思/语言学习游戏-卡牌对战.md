# 语言学习竞技卡牌游戏 · 产品构思文档

> 整理自 2026-04-30 的头脑风暴对话

---

## 一、市场现状分析

### 1.1 视频学外语 + 闯关机制的APP

全球范围内，真正把"视频内容"和"闯关机制"都做到位的产品很少：

| 产品 | 视频内容 | 闯关机制 | 评价 |
|---|---|---|---|
| **FluentU** | ✅ 真实视频（电影预告、MV、新闻） | ✅ 视频后测验，按难度分级 | 最接近"视频+闯关"的完整形态 |
| **Lingopie** | ✅ 电视剧、电影 | ⚠️ 游戏化功能偏弱 | 更像沉浸式看剧+辅助工具 |
| **Jolii** | ✅ Netflix、YouTube | ⚠️ 有进阶路径但关卡感不强 | AI驱动，比较新 |
| **Sabi** | ✅ Netflix、YouTube | ⚠️ 游戏化是辅助层 | Chrome扩展，不是独立APP |
| **Duolingo** | ❌ 核心不是视频 | ✅ 做得最成熟 | 短句练习+听力为主 |

### 1.2 为什么没人挑战多邻国？

多邻国存在这么多年，不是没人挑战，是挑战了但打不过。原因：

1. **核心壁垒不是"教得好"，是"用得久"** — 多邻国本质上是一个游戏化引擎套了个语言学习壳，streak、联赛、XP、heart机制把日活拉到恐怖的数字
2. **免费策略** — 用免费把门槛砸没，等规模起来后后来者在成本结构上打不过
3. **品牌=品类** — "学外语app"这个品类里多邻国已是默认选项
4. **竞争维度不对** — 打不过它的不是因为产品差，是因为竞争不在"谁教得好"上，而是在"谁能让用户每天点开"上

**核心矛盾：** 多邻国收割的是那90%"想学但坚持不了"的用户，这个市场才是最大的。

## 二、多邻国的本质洞察

### 2.1 99%做专业，1%做有趣

- 99%的教育公司在卷"教得深"——课程体系、教学法、CEFR分级、语法覆盖率
- 多邻国在做"学得久"——解决的核心问题是"怎么让你明天还来"
- **认知差：** 传统教育思维"内容为王"，多邻国证明了C端市场"习惯为王"

### 2.2 多邻国的游戏化天花板

- 多邻国的"有趣"是**浅层有趣**——streak焦虑、卡通画风、排行榜竞争
- 和真正的游戏相比（《原神》构建开放世界叙事、《塞尔达》涌现式玩法），多邻国的游戏化在真正的游戏面前"屁都不是"
- 多邻国让人"打卡"，但没有让人"沉浸"
- 学语言这个行为本身（点选答案、翻译句子）还是像做题

### 2.3 用户心态的根本区别

| | 多邻国 | 理想产品 |
|---|---|---|
| 用户打开时的想法 | "我来学一下" | "我来玩一下" |
| 身份 | 学生 | 玩家 |
| 驱动力 | 义务感（怕断streak） | 期待感（想看后面发生什么） |
| 义务感 | 会消耗 | — |
| 期待感 | — | 会增长 |

## 三、产品核心理念

### 3.1 一句话定义

> **不是"学习APP加了游戏元素"，而是"游戏APP碰巧教你东西"。**

出发点完全不同：
- 前者是教育公司思维，用户进来心态是"我要学习了"（有心理负担）
- 后者是游戏公司思维，用户进来心态是"我要玩了"（没有负担，只有期待）

### 3.2 成功先例

- 《大富翁》教你算数
- 《模拟城市》教你城市规划
- 《文明》教你历史
- 没人打开《文明》想的是"我来学历史"，但玩了500小时后对世界史的了解比上一学期课都深

### 3.3 核心设计原则

**先有一个好玩的世界，语言是这个世界里的氧气。**
- 你不会为了"学英语"去玩
- 你会因为想通关、想探索、想赢，而不得不去理解这些语言
- 等你反应过来的时候，你已经在用英语思考了

## 四、游戏形式探索

### 4.1 竞技是核心上瘾机制

竞技的吸引力：
- 对手是真人，不可预测——背题没用，必须真正掌握
- 赢了有真实满足感——不是虚拟奖章，是真的赢了一个人
- 输了有真实挫败感——比任何push notification都有效
- 有社交传播性——"我今天钻石段了"能自发传播
- 有赛季制——每赛季重置，永远有新目标

**多邻国的排行榜是假竞技（数字排名），真正的竞技是实时博弈、你死我活。**

### 4.2 具体竞技形态

| 形态 | 机制 | 语言技能覆盖 |
|---|---|---|
| 实时卡牌对战 | 双方同时出牌，读懂卡牌才能用好 | 阅读为主 |
| 辩论对战 | 双方轮流用目标语言表达，AI评判 | 口语+逻辑 |
| 社交推理 | 狼人杀变体，所有交流用目标语言 | 听说读写全覆盖 |
| 生存淘汰 | 大逃荒，语言任务淘汰末位 | 综合能力 |
| 场景抢答 | 同一场景，比谁先答对 | 听力+口语+反应 |

### 4.3 最终选定：策略竞技卡牌

选择卡牌对战的原因：
- 策略游戏天然给思考时间，语言理解本身也需要时间
- 开发成本低、对美术要求低、但重玩性极高
- 是最容易冷启动的游戏品类

## 五、卡牌对战系统设计

### 5.1 核心玩法

- 两个玩家各持一套卡牌
- 卡牌效果描述用目标语言写成
- **你必须读懂卡牌才能知道它有什么效果**
- 同一张牌，语言能力决定了你用它的水平

### 5.2 卡牌类型

**① 战斗牌（负责"好玩"）**
```
[Quick Punch]
Do 3 damage now. Draw 1 card.
```

**② 场景牌（负责"学习"）——触发真实生活场景**
```
[场景牌 · 餐厅]
You walk into a small café. A waiter says:
"Good evening, Do you have a reservation?"

Choose:
A) "Table for two, please. Can we sit by the window?"
B) "I want eat food now."
C) "Where is the bathroom?"
D) (say nothing and walk out)
```

**③ 暗牌（高风险高回报）**
- 效果用目标语言写成，没有翻译
- 读懂了威力巨大，读错了浪费一个回合
- 创造策略抉择：你敢不敢赌自己的语言能力

**④ 陷阱牌（对抗性）**
```
[Word Trap]
Your opponent reads one short sentence. 
If they understand it right, nothing happens. 
If they get it wrong, they lose 2 cards.
```

### 5.3 段位与语言难度映射

| 段位 | 语言难度 | 词汇量 | 设计意图 |
|---|---|---|---|
| 铜-银 | A1-A2 | ~500-1500词 | 明牌为主，暗牌少，新手友好 |
| 金-铂 | A2-B1 | ~1500-3000词 | 暗牌增多，开始需要真正读懂 |
| 钻石+ | B1-B2 | ~3000-5000词 | 暗牌成为核心，语言能力决定胜负 |
| 王者 | B2+ | 5000+词 | 限时出牌+复杂文本，压力下的理解 |

**段位越高，语言的权重越大。能在高段位存活的人，语言能力一定不差。**

## 六、学习机制设计（最关键的部分）

### 6.1 问题：不能只靠读卡牌文字

纯靠读卡牌 = 阅读理解练习。听说写全丢了，学出来的结果是能看懂但听不懂、说不出、写不对。

### 6.2 解法：每张场景牌是一次多感官沉浸体验

不是看到一行英文句子，是**进入一个场景**——有声音、有画面、有说话、有打字、有决策。

**四技能同时卷入：**
1. **听力** — 听懂对方说了什么（语音播放）
2. **阅读** — 读懂选项/卡牌描述
3. **口语** — 按下麦克风把选项大声说出来（语音识别）
4. **写作** — 打字回答/拼写答案

### 6.3 场景牌保持竞技感的方案

**场景不是"你一个人做题"，是两个人同时进入同一个场景，比谁更快更准。**

- **听力抢答：** 两人同时听到同一段话，谁先答对谁得分
- **口语PK：** 两人同时回应同一个场景，AI评分谁的回应更好
- **打字竞速：** 同一段广播信息，谁先提取关键信息并打出来
- **开放式对决：** 没有选项，用自己的话说，AI评估内容质量

### 6.4 游戏节奏设计

```
回合1: 出战斗牌（快节奏，读卡、决策、出牌）     ← 纯竞技
回合2: 场景牌触发（听+说+读+写，和对手抢答）    ← 学习+竞技
回合3: 出战斗牌（继续策略博弈）                 ← 纯竞技
回合4: 场景牌触发（新的真实场景PK）             ← 学习+竞技
回合5: 决战回合（战斗牌+场景牌混合，赛点）       ← 全力竞技
```

快节奏和慢节奏交替，像音乐里的节拍变化。

## 七、词汇覆盖方案

### 7.1 问题：游戏词汇≠真实词汇

"do 3 damage"、"draw 1 card"这些词现实中几乎永远用不到。

### 7.2 解法：先确定词汇表，再反向设计场景

```
CEFR A1 词汇表（约500词）
    ├── 日常类（150词）→ 设计"餐厅""购物""问路"场景牌
    ├── 社交类（100词）→ 设计"派对""电话""邻居"场景牌
    ├── 情感类（80词）→ 设计"吵架""表白""安慰朋友"场景牌
    ├── 旅行类（100词）→ 设计"入住""过海关""迷路"场景牌
    └── 职场类（70词）→ 设计"入职""开会""写邮件"场景牌
```

**每一个词汇都有明确的出现场景，不是随机塞进去的。**

| 语言等级 | 词汇量 | 场景牌数量 | 覆盖场景 |
|---|---|---|---|
| A1 | ~500词 | 80张 | 基础生存：点餐、购物、问路、打招呼 |
| A2 | ~1500词 | 120张 | 日常生活：看医生、打电话、约朋友、坐车 |
| B1 | ~3000词 | 150张 | 社交职场：面试、会议、租房、社交活动 |
| B2 | ~5000词 | 120张 | 深度交流：辩论、讲故事、表达观点、谈判 |

## 八、长期记忆设计

### 8.1 间隔重复藏进游戏循环

每个玩家有"记忆档案"，系统默默记录每个词的掌握程度：

```
"reservation"    → 7天前学会，对战中用过3次 → 记忆度 80%
"platform"       → 3天前遇到，只见过1次 → 记忆度 40% ⚠️ 快忘了
"I can't believe" → 今天刚学 → 记忆度 20%
```

**当记忆度下降到临界点，这个词会在下一场对战中"自然"出现——不是弹出复习页面，是出现在新的场景牌里。**

### 8.2 记忆强度层次

| 游戏行为 | 记忆科学原理 | 记忆强度 |
|---|---|---|
| 新卡牌解锁 | 首次学习（编码） | ★☆☆☆☆ |
| 对战中遇到 | 第一次检索练习 | ★★☆☆☆ |
| 3天后再次遇到 | 间隔重复 | ★★★☆☆ |
| 在不同场景里遇到 | 情境迁移 | ★★★★☆ |
| 开放式口语中用出来 | 主动输出 | ★★★★★ |
| 在竞技中靠它赢了 | 情绪记忆 | ★★★★★+ |

**认知科学证明：情绪越强的记忆越持久。** 在对战中用一个词绝杀对手的那个瞬间，这个词一辈子忘不了。

### 8.3 学习循环

```
拿到新卡 → 解锁读牌30秒（首次接触）
    → 带着新卡去对战
        → 看懂了 → 记忆强化
        → 没看懂吃亏了 → 复盘讲解
    → 练习房巩固（主动记忆）
    → 下次对战用出来（真正掌握）
```

## 九、冷启动策略

### 9.1 全球市场是天然优势

- 地球上永远有人醒着——中国凌晨3点是美国下午2点
- 全球覆盖天然解决了"某个时段没人"的问题

### 9.2 前期AI替代真人

PUBG手游早期70%是bot，但玩家根本分不出来。同理：
- AI对手起真人风格的名字
- 表现水平参差不齐（有犯错、反应慢、犯蠢的时候）
- 偶尔用表情、偶尔打错字
- 段位分布自然

### 9.3 四步冷启动

| 阶段 | 用户量 | 竞技形态 | 用户感知 |
|---|---|---|---|
| 冷启动 | 0-1万 | AI对手+异步挑战+每日排行榜 | "我每天跟人比拼，挺刺激" |
| 增长期 | 1-10万 | 定时赛事+好友PK | "我拉朋友来跟我打" |
| 成熟期 | 10万+ | 实时匹配+段位联赛 | "这是真正的竞技游戏" |

### 9.4 异步竞技

不需要同时在线也能打：
- **影子对战：** 挑战别人昨天的记录
- **好友PK链接：** 分享链接就能挑战
- **每日挑战赛：** 全服同一题，看排行榜

### 9.5 时区分割

首发市场选3-4个语言圈，每个圈先建密度：
- 学英语 → 先攻东南亚（菲律宾、越南、印尼）
- 学日语 → 先攻中国和东南亚
- 学韩语 → 先攻东南亚（K-pop粉丝）
- 学中文 → 先攻日韩泰

### 9.6 KOL引爆

找语言学习YouTuber/TikToker直播对战：
"两个美国博主比赛谁的日语更好" → 天然爆款内容

## 十、与多邻国的根本对比

| 维度 | 多邻国 | 本产品 |
|---|---|---|
| 产品基因 | 教育公司+游戏元素 | 游戏公司+教育效果 |
| 用户动机 | 怕断streak（义务感） | 想赢对手/想看剧情（期待感） |
| 语言角色 | 被考核的科目 | 游戏世界的生存资源 |
| 反馈来源 | 虚拟奖励（XP、宝石） | 真实对战胜负+语言理解带来的优势 |
| 上瘾机制 | 行为主义（斯金纳箱） | 竞技驱动（胜负欲）+叙事驱动（好奇心） |
| 学习覆盖 | 词汇为主，口语弱 | 听说读写四技能全覆盖 |
| 记忆方式 | 重复刷题 | 间隔重复+情绪锚定+情境迁移 |
| 难度感知 | "英语题变难了" | "游戏变难了" |
| 技能迁移 | 学了不太会用 | 场景牌全是真实生活场景 |

## 十一、核心金句

> "多邻国是学习app加了游戏元素，我们要做的是游戏app碰巧教你东西。"

> "用户每次打开App前想着的是来玩一下，不是来学一下。"
