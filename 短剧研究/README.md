# 短剧研究

> 为短剧出海业务做知识储备和能力积累
> 方向：**英文短剧，面向海外市场**（TikTok / YouTube Shorts / Instagram Reels）

## 目录结构

- `行业分析/` — 行业趋势、市场规模、商业模式
- `内容策划/` — 选题、剧本结构、叙事技巧
- `用户研究/` — 用户画像、观看行为、留存策略
- `案例拆解/` — 成功短剧的拆解分析
- `制作流程/` — 拍摄、剪辑、后期等制作知识（AI 工具链）
- `参考资料/` — 原始文章、报告、素材存档

## 出海方向核心差异

| 维度 | 国内短剧 | 英文出海短剧 |
|------|----------|-------------|
| 平台 | 红果、抖音、快手 | TikTok、YouTube Shorts、Instagram Reels |
| 语言 | 中文 | 英文（美式/英式根据目标市场） |
| 付费模式 | 付费解锁、会员订阅 | 广告分成为主 + 品牌合作 + 付费订阅 |
| 时长 | 1-3 分钟/集 | 30s-90s/集（TikTok 偏短，YouTube 可稍长） |
| 受众画像 | 国内下沉市场、女性为主 | 海外年轻用户（18-34）、男女均衡 |
| 热门题材 | 霸总、重生、逆袭、甜宠 | Thriller、Horror、Romance、Comedy、Mystery |
| 审核 | 国内平台审核（内容合规） | 海外平台审核（版权、社区规范） |
| AI 工具 | 即梦、豆包 | ChatGPT、MidJourney、ElevenLabs、Runway、HeyGen |
