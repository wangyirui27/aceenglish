# 英文短剧出海指南

> 整理时间：2026-04-29
> 方向：英文短剧，面向 TikTok / YouTube Shorts / Instagram Reels

---

## 一、为什么做出海短剧

- TikTok 全球月活超 20 亿，短视频内容需求巨大
- YouTube Shorts 基金 + 广告分成，变现路径成熟
- 海外短剧市场仍处于**早期红利阶段**，竞争远不如国内激烈
- AI 工具降低了制作门槛，一个人就能做出专业级内容

---

## 二、目标平台对比

| 平台 | 最佳时长 | 竖屏/横屏 | 变现方式 | 核心算法偏好 |
|------|----------|-----------|----------|-------------|
| TikTok | 30s-90s | 9:16 竖屏 | Creator Fund + 品牌合作 + 直播打赏 | 完播率 > 互动率 > 关注率 |
| YouTube Shorts | 30s-60s | 9:16 竖屏 | 广告分成 + 频道会员 | 点击率 > 完播率 > 订阅转化 |
| Instagram Reels | 15s-60s | 9:16 竖屏 | 品牌合作 + Shopping | 分享率 > 保存率 > 评论率 |

**建议**：先集中做一个平台（推荐 TikTok），跑通后再多平台分发。

## 三、海外热门短剧题材

### 高流量题材（已验证）

| 题材 | 英文关键词 | 说明 |
|------|-----------|------|
| 悬疑/反转 | Mystery, Plot Twist, Unexpected Ending | "Watch till the end" 类型，完播率极高 |
| 恐怖/惊悚 | Horror, Scary, Creepy, Jump Scare | TikTok 上传播极快，适合 30s-60s 格式 |
| 爱情/虐恋 | Romance, Love Story, Heartbreak | 全球通用题材，情感共鸣强 |
| 喜剧/反转 | Comedy, Funny, Relatable | 制作门槛最低，传播最快 |
| 逆袭/复仇 | Revenge, Payback, Karma | 国内验证的"爽感"逻辑海外同样有效 |
| 科幻/脑洞 | Sci-Fi, What If, Mind-Blown | 差异化题材，竞争较少 |

### 不推荐的题材

- 霸总/甜宠（国内强，海外弱——文化差异大）
- 重生/系统流（需要大量旁白解释设定，海外观众不买账）
- 家族内斗（过于复杂，1 分钟讲不清楚）

## 四、英文短剧叙事结构（1 分钟版）

海外观众的注意力更短、对"废话"的容忍度更低。推荐结构：

```
0-3s    HOOK — 必须在 3 秒内抓住注意力
        画面冲击 / 悬念问题 / 冲突场景
        ❌ 绝对不要用 "Hey guys, welcome to..."
        
3-10s   SETUP — 快速建立情境（不超过 2 句台词）

10-25s  CONFLICT — 矛盾升级，节奏加快

25-45s  TWIST — 反转揭露，情绪爆发

45-55s  RESOLUTION / CLIFFHANGER —
        选项A：短篇 → 给一个出人意料的结局
        选项B：系列 → 留钩子，引导看下一集

55-60s  ENDING CARD — "Follow for Part 2" / "What happens next?"
```

### 核心原则

1. **Show, don't tell** — 用画面讲故事，少用旁白
2. **Conflict first** — 第一帧就是冲突，不要铺垫
3. **Dialogue < 5 words per line** — 台词简短有力，口语化
4. **Every second counts** — 1 秒都不能浪费，删掉所有空镜

## 六、英文剧本提示词模板

### 用 ChatGPT 生成 1 分钟英文短剧剧本

```
Write a 1-minute short drama script for TikTok in English.

Genre: [Thriller / Horror / Romance / Comedy / Mystery]
Logline: [一句话故事梗概]

Requirements:
1. HOOK in the first 3 seconds — start with a shocking image, 
   provocative question, or conflict. NO intro, NO "hey guys".
2. Show, don't tell — use visual storytelling, minimize narration.
3. Dialogue must be natural, conversational American English. 
   Keep lines under 8 words each.
4. One major TWIST at the 30-second mark.
5. End with a cliffhanger that makes viewers want Part 2.
6. Total 10-12 shots, each shot 5-6 seconds.
7. Format each shot as:
   SHOT # [timecode]: [camera angle], [action], [dialogue]
   
8. At the end, include a shot list with AI image generation prompts 
   for each shot (in English, photorealistic style, with lighting 
   and material descriptions).
```

### 用 ChatGPT 生成角色设定提示词

```
Based on the script above, create AI image generation prompts 
for each character's character design sheet.

For each character, write a prompt that includes:
1. Physical appearance (age, ethnicity, build, facial features)
2. Clothing and accessories
3. Distinguishing marks (tattoos, scars, etc.)
4. PBR material descriptions (skin texture, fabric details)
5. Lighting setup
6. Format: full body shot, neutral background, photorealistic,
   Kodak Portra 400 style, 4K

Write prompts in English for use with MidJourney.
```

## 七、发布策略

### 标题公式

```
[悬念] + [情绪钩子] + [系列标记]
```

示例：
- "She found a letter in her dead husband's jacket... Part 1"
- "He didn't know his girlfriend was filming. Plot twist."
- "I caught my roommate doing THIS at 3am"

### 标签策略

```
#shortdrama #shortfilm #storytime #plottwist #viral
#filmmaking #cinematic #suspense #horrorstory
+ 题材相关标签 + 趋势标签
```

### 发布时间（UTC）

| 目标市场 | 最佳发布时间 |
|----------|-------------|
| 美国 | 12:00-15:00 EST（北京时间 01:00-04:00） |
| 英国 | 12:00-14:00 GMT（北京时间 20:00-22:00） |
| 东南亚 | 19:00-21:00 ICT（北京时间 20:00-22:00） |

### 系列化策略

- 做**系列短剧**（Part 1, Part 2, Part 3...）而非单集
- 每集结尾留钩子，引导关注+等下一集
- 系列完播率 > 单集完播率，算法权重更高

### 平台分成参考

| 平台 | 分成方式 | 大概收入 |
|------|----------|----------|
| TikTok Creator Fund | 按播放量 | $0.02-0.04 / 1000 views |
| YouTube Shorts | 广告分成 | $0.01-0.07 / 1000 views |
| YouTube 长视频 | 广告分成 | $3-8 / 1000 views（CPM） |

**关键认知**：平台分成收入很低，真正的收入来自**品牌合作**和**自有产品**。短剧是流量入口，不是最终产品。
