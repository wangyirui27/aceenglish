# 🧾 The Bill · 文生视频提示词（POV 版）

> 共 **19 段**唯一切片，每段 15-45 秒  
> 全部 POV 视角 + 环境音静音  
> 场景：美式餐厅，暖黄灯光，卡座  
> 角色：👩‍🍳 Server（女，20+）、👨‍💼 Manager（男，40+）

---

## 1. 开场 · Server 递账单

```
POV视角，环境音静音。美式餐厅，暖黄灯光，面前是餐桌，两个空盘和杯子在桌上。女服务生从对面走过来，把账单夹放在桌上，微笑看着镜头。台词："Here's your bill. Whenever you're ready."
```

---

## 2. Route A · 礼貌质疑

```
POV视角，环境音静音。视线从账单上抬起来，看向面前站着的女服务生，画面微微上移。说话语气平静，略带困惑。台词："Excuse me, I think there's a mistake — we didn't order any wine."
```

---

## 3. Route A · Server 回应

```
POV视角，环境音静音。服务生接过账单（手从画面下方伸入取走），皱眉低头看了一眼账单，然后抬头看向镜头，表情不确定但不凶。台词："Let me check... Hmm, it says here — one bottle of house red. Are you sure you didn't order it?"
```

---

## 4. A1 · 主角举证据

```
POV视角，环境音静音。面前是服务生，一只手从画面下方伸入镜头，掰着手指数——先伸出一根手指，再两根。说话语气坚定但友好。台词："Yes, I'm sure. We had two sodas and two pasta dishes. That's it. Could you take it off the bill, please?"
```

---

## 5. A1 · Server 认错（🏆）

```
POV视角，环境音静音。服务生低头查了系统（在柜台设备上点击了几下），然后抬头看向镜头，表情从困惑变成恍然大悟，不好意思地笑了笑。台词："You're right — that was for table 6. I'm so sorry. Let me fix it."
```

---

## 6. A2 · 主角坚定升级

```
POV视角，环境音静音。视线平视面前的服务生，说话语气比刚才更认真但依然礼貌，画面稳定不晃动。台词："I'm absolutely sure. Can I speak to the manager?"
```

---

## 7. A2 · Manager 处理（🏆）

```
POV视角，环境音静音。男经理从餐厅深处走过来，手里拿着平板，低头看了一眼屏幕，然后抬头对着镜头说话，表情专业但友善。台词："I just checked — it was the wrong table. I've taken it off. Sorry about that."
```

---

## 8. Route B · 含糊抱怨

```
POV视角，环境音静音。视线低垂看着桌上的账单，一只手从画面下方伸入——手指点在账单上的$45那一行。然后视线抬起来，犹豫地看向面前的服务生。台词："Excuse me... this doesn't look right. What is this $45?"
```

---

## 9. Route B · Server 防卫

```
POV视角，环境音静音。服务生双手交叉在胸前，下巴微抬，直视镜头，语气带着防卫。台词："That's the house wine. You ordered it — I brought it to your table myself."
```

---

## 10. B1 · 主角坚持立场

```
POV视角，环境音静音。视线微微左右摇头（画面轻微左右晃动），面对服务生，语气简短直接但不大声。台词："No. I didn't order wine. We only had sodas. Please check again."
```

---

## 11. B1 · Server 不情愿认错（🏆）

```
POV视角，环境音静音。服务生叹了口气，不情愿地转身在柜台设备上查系统（侧脸或背影），几秒后转回来面对镜头，表情有点尴尬。台词："...Fine. Let me check. ...Oh. It was table 7. Sorry."
```

---

## 12. B2 · 主角退缩

```
POV视角，环境音静音。视线垂下，避开面前服务生的目光，画面看向桌面——手指无意识地摸着账单边缘。说话声音变小，底气不足。台词："Are you sure? I mean... maybe I forgot. Never mind."
```

---

## 13. B2 · Server 确认收钱（💀）

```
POV视角，环境音静音。服务生站在面前，表情没变，嘴角微微上扬，眼神里闪过一丝"果然如此"。她拿出刷卡机递向镜头。台词："Yeah, I'm sure. So... cash or card?"
```

---

## 14. Route C · 直接指控

```
POV视角，环境音静音。一只手从画面下方啪地把账单拍在桌上，视线随即抬起来瞪向面前的服务生。说话声音比正常大，语气愤怒。台词："This is wrong. We never ordered wine. What is this?"
```

---

## 15. Route C · Server 被冒犯

```
POV视角，环境音静音。服务生往后退了半步（画面中她变小了一点），先愣了一下，然后表情冷下来，双臂交叉，直视镜头，语气带刺。台词："I brought it to your table. You didn't say anything then. Now you don't want to pay for it?"
```

---

## 16. C1 · 主角强制降火

```
POV视角，环境音静音。深吸一口气的声音，两只手从画面下方平放在桌上，画面稳定。说话声音放低，语速放慢，努力控制情绪。台词："Look — I'm not saying I won't pay. I'm saying it's not ours. Can we just check the order?"
```

---

## 17. C1 · Server 冷脸修改（⚠️）

```
POV视角，环境音静音。服务生冷着脸，不跟镜头对视，转身在平板上点了几下（侧身/背影），语气冷淡。台词："...It was a mistake. I'll fix it."
```

---

## 18. C2 · 主角火上浇油

```
POV视角，环境音静音。画面突然升高（站起来了），一只手从下方伸入镜头指着桌上的账单。背后几桌客人转头看过来。说话声音很大，语气愤怒，完全失去耐心。台词："I'm not paying for something I didn't order. This is ridiculous."
```

---

## 19. C2 · Manager 赶人（💀）

```
POV视角，环境音静音。男经理快步走到桌前，面对镜头，表情严肃，手臂抬起指向门口方向。背景里几桌客人都朝这边看。语气不容商量。台词："I've asked you to leave. Pay your bill and go."
```

## 📊 制作速查

| 场景 | 角色 | 段数 | POV 备注 |
|---|---|---|---|
| 卡座 + Server | 👩‍🍳 | 14 段 | 主角不可见，手偶尔入镜 |
| 卡座 + Manager | 👨‍💼 | 2 段 | A2、C2 各一 |
| Server 单独 | 👩‍🍳 | 2 段 | 开场 + 查系统 |
| 站立 POV | 👩‍🍳👨‍💼 | 1 段 | C2 唯一站立 + 手入镜 |

> **POV 规则**：全程镜头 = 主角眼睛。主角不可见。主角的手偶尔从画面下方伸入（点账单/摆手/拍桌）。主角的"摇头"用画面轻微晃动替代。
