## 一、产品定位

一款 **AI 英语听力学习 APP**，将美剧和电影通过 AI 剪辑成完整对话片段（每段≤2分钟），以**短视频信息流**的形式呈现（类似抖音的刷视频体验），让用户在沉浸式刷剧中提升英语听力和综合能力。

## 三、内容体系

### 3.1 视频内容
- 来源：美剧、电影
- AI 剪辑成独立片段，每段是一个**完整对话**，时长≤2分钟
- 所有视频有**难度等级**：1星～5星，对应 A1～C1，C1 以上统一定义为 5 星

### 3.2 每条视频的三屏结构

| 屏 | 内容 | 说明 |
|----|------|------|
| **第一屏：视频** | 视频播放 + 闪答 + 字幕 + 重遇弹幕 | 核心消费场景 |
| **第二屏：学习笔记** | AI 总结的知识点 | 提炼视频中重要的短语和知识点 |
| **第三屏：跟读** | 逐句台词列表 | 点击任意一句→单句循环播放→用户点按钮跟读→跟读评分 |

## 四、核心功能

### 4.1 短视频流
- 类抖音的上下滑动刷视频体验
- 视频流中会**穿插微练习题**

### 4.2 闪答
- 视频播放到指定位置时，弹出迷雾遮罩
- 2 选 1：听到了哪个单词/短语
- 答对有积分奖励
- 难度较低——只要专心听循环播放的闪答片段，总能答对（除非不用心）
- 设计意图：保持用户专注度

### 4.3 创新字幕系统（4 种模式）

| 模式 | 说明 |
|------|------|
| **减压字幕** | 只显示英文，但在超纲词（超过 A2 水平的单词）后面附中文释义。一句话可能只有一个词加了翻译 |
| **中英字幕** | 中英双语字幕 |
| **纯英文字幕** | 仅英文 |
| **延迟字幕** | 所有字幕延迟画面 2 秒后显示 |

### 4.4 全局 AI 教练
- 用户观看视频时，任意一句台词都可以点击"AI 一键解析"
- 弹出全局 AI 教练窗口（聊天形式），结合上下文解答台词含义和知识点
- ⚠️ **当前问题**：解答质量较低，AI 生成内容质量未达标（产品负责人标准较高）

### 4.5 全局字典 + 框选查词
- APP 内任何位置的文字（台词、学习笔记、AI 教练对话）都可以**框选**
- 框选后弹出字典查询
- 查到的单词/短语可以**收藏到收藏夹**
- 字典结果页也可以呼唤全局 AI 教练进一步解答

### 4.6 智能搜索
- 搜索框输入单词后，搜索结果分三层：
  1. **字典内容**：该单词的释义
  2. **用法展示**：该单词的不同用法
  3. **视频片段**：该单词/短语在现有哪些视频片段中出现
- 点击搜索结果中的视频→**只播放包含该单词的那句台词**，单句循环
- 上划切换到下一个包含该单词的视频，同样单句循环播放

### 4.7 重遇（弹幕功能）
- 用户观看视频时，如果当前视频出现了用户**收藏过的单词**
- 视频上飞出弹幕，告诉用户："你收藏过的这个单词是第 X 次出现了"
- 功能名称：**重遇**

### 4.8 微练习（6 种题型）

| # | 题型 | 说明 |
|---|------|------|
| 1 | 预测完型 3 选 1 | — |
| 2 | 乱序单词拖拽拼句 | — |
| 3 | 8 词选 4 | 一段视频里 8 个词中选出听到的 4 个 |
| 4 | 选最合适的表达 | 看视频回答问题，选出最合适的表达 |
| 5 | 选视频主要内容 | 看视频选出主要内容 |
| 6 | 选台词含义 | 看视频选出这句台词的含义 |

微练习出现在：
- **视频流中随机插入**（刷视频时突然出一题）
- **收藏夹 → 每日守护**
- **收藏夹 → 考一下**（针对单个单词）
- **收藏夹 → 学习列表**（见 4.9）

> **[2026-04-27 确定]** 微练习出题范围包含用户收藏的单词，答题结果直接影响该单词的熟悉度等级（答对升级、答错不变或降级）。微练习不再只是独立练习，而是熟悉度体系的核心推进手段。

### 4.9 收藏夹 & 单词管理

- 收藏夹位置：**视频流顶部标签**，且是打开 APP 的**第一屏**
- 收藏内容：单词、语块
- 熟悉度分级：**生词 → 眼熟 → 熟悉 → 精通**
- 复习机制（间隔重复）：
  - 答对升一级
  - 隔天才能再升级
  - 7～14 天不复习降一级
  - 每天最多面对 5 个降级单词
  - 答对就保住等级
- **每日守护**：每天提醒用户守护 5 个即将降级的单词（做微练习题）
- **考一下**：在收藏夹点开某个单词，针对该单词做一道微练习

#### 导入辞书 & 过滤 [2026-04-27 确定]

- 用户可导入整本辞书（如四六级词汇、雅思词汇等），一次性导入数千单词
- 导入时允许用户**一键过滤掉 A1、A2 级别的基础单词**，只保留有学习价值的词
- 过滤后的单词进入待学习队列

#### 学习列表 [2026-04-27 确定]

- 收藏夹新增**「学习列表」**标签页
- 用户可设置**每月学习目标**（如本月学 100 个单词）
- 系统从待学习队列中自动填充对应数量的单词到学习列表
- 本月任务：**将学习列表中的所有单词推到「精通」**
- 未在当月完成的单词**累计到下个月**
- 用户可**随时调整本月学习目标数量**
- 学习列表中的单词通过微练习、每日守护、考一下等方式推进熟悉度

### 4.10 跟读 & 评分
- 第三屏列出当前视频所有台词
- 点击任意一句→单句循环播放
- 用户点击按钮开始跟读→系统评分

### 付费功能分层（v0.6 终版方案）

> 详细讨论见 DISCUSSIONS/2026-04-26_付费功能分层_part1~4
> 核心理念：**免费给"体验"，收费给"体系"**

| 功能 | 免费 | 付费 |
|------|------|------|
| 刷视频（无限量） | ✅ | — |
| 闪答互动 | ✅ | — |
| 纯英文/中英字幕 | ✅ | — |
| **减压字幕** | ✅（改为免费，作为差异化核心体验） | — |
| 延迟字幕 | ❌ | ✅（可自定义 2/3/5 秒） |
| 基础字典查词+框选 | ✅ | — |
| 收藏单词 | ✅ **不限量**（改：不卡收藏，卡复习） | — |
| AI 教练 | 每日 3 次（用完前两行模糊化） | 无限次 |
| 学习笔记 | 前 2 条知识点 | 完整内容 |
| 跟读（听+念） | ✅ | — |
| 跟读评分 | ❌ | ✅ 评分+定位+建议 |
| 搜索词典+用法 | ✅ | — |
| 搜索→视频语境 | 每次 3 条 | 无限+难度筛选 |
| 难度筛选视频 | ❌（跟算法推荐） | ✅ |
| 每日守护+复习系统 | ❌ | ✅ 遗忘曲线/统计/计划 |
| 微练习 | 每日 3 题（用完选项模糊） | 无限 |
| 重遇弹幕 | ✅ **不限次**（防噪靠设计不靠上限） | 点击后动作：加复习队列/加练/周报 |

### 新用户转化路径
- Day 0：全免费体验（减压字幕+闪答+查词） → 感受差异化
- Day 1-3：建立"刷+查+收藏"习惯
- Day 3-7：多个付费触发点集中出现 → **此时推 3 天免费全功能体验**
- 体验结束 → 用户已有积累，转化为"要不要继续"

### 收费前必须补上：轻量能力自适应
- 新用户引导问 2 个问题定位水平
- 之后用闪答/微练习正确率动态调难度
