## 一、产品定位

**头号听力**是一款 AI 英语听力学习 APP。将美剧和电影通过 AI 剪辑成完整对话片段（每段 ≤2 分钟），以**短视频信息流**形式呈现（类抖音的上下滑动刷视频体验），让用户在沉浸式刷剧中提升英语听力和综合能力。

**一句话**：刷剧学英语，像刷抖音一样上瘾。

## 三、内容体系

### 3.1 视频内容
- **来源**：美剧、电影
- **加工**：AI 剪辑成独立片段，每段是一个**完整对话**，时长 ≤2 分钟
- **难度分级**：1 星～5 星，对应 A1～C1，C1 以上统一定义为 5 星

### 3.2 每条视频的三屏结构

| 屏 | 内容 | 说明 |
|----|------|------|
| **第一屏：视频** | 视频播放 + 闪答 + 字幕 + 重遇弹幕 | 核心消费场景 |
| **第二屏：学习笔记** | AI 总结的知识点 | 提炼视频中重要的短语和知识点 |
| **第三屏：跟读** | 逐句台词列表 | 点击任意一句→单句循环播放→用户点按钮跟读→跟读评分 |

## 四、全部功能

### 4.1 短视频流
- 类抖音的上下滑动刷视频体验，全屏沉浸式设计，深色主题
- 视频流中可**主动触发微练习**（页面有微练习按钮）
- **视频完整播放后自动插入微练习**：用户看完一条视频后，自动弹出该视频的微练习（默认第一题为"这个视频描述了什么"）
- **底部标签内容每日自动变化排序**：让视频流每天看起来有新鲜感，避免静态感
- **评论区改为笔记区**：视频下方不叫"评论"，而是学习笔记讨论区
- **无封面视频自动生成封面图**：运营上传视频时无需手动设置封面
- **金币显示**：视频流页面展示金币数量，字体统一为 Saira

### 4.2 闪答
- 视频播放到指定位置时，弹出迷雾遮罩
- **2 选 1**：听到了哪个单词/短语
- 答对有金币奖励
- **答完后两个选项显示中文翻译**（名词优先），帮助用户即时理解
- 难度较低——只要专心听循环播放的闪答片段，总能答对
- 设计意图：保持用户专注度，防止刷视频时走神

### 4.3 创新字幕系统（4 种模式）

| 模式 | 说明 | 免费 |
|------|------|------|
| **减压字幕** | 只显示英文，超纲词（超过 A2 水平）后附中文释义。一句话可能只有一个词加了翻译 | ✅ |
| **中英字幕** | 中英双语字幕 | ✅ |
| **纯英文字幕** | 仅英文 | ✅ |
| **延迟字幕** | 字幕延迟画面 2 秒后显示，付费用户可自定义 2/3/5 秒 | 付费 |

### 4.4 全局 AI 教练
- 任意一句台词点击"AI 一键解析"，弹出聊天式 AI 教练窗口
- 结合上下文解答台词含义、语法、文化背景、使用场景
- 支持追问，可连续对话
- ⚠️ 当前问题：解答质量未达标（产品负责人标准较高），v0.6 考虑先以 Beta 标签上线
- **免费用户**：每日 3 次，第 4 次起输出前两行后模糊化，显示"解锁完整解析"
- **付费用户**：无限次

### 4.5 全局字典 + 框选查词
- APP 内任何位置的文字（台词、学习笔记、AI 教练对话）都可以**框选**
- 框选后弹出字典查询（释义、音标、词性、例句、难度等级）
- 查到的单词/短语可**收藏到收藏夹**
- 字典结果页可呼唤 AI 教练进一步解答

### 4.6 智能搜索
搜索框输入单词后，结果分三层：
1. **字典内容**：单词释义
2. **用法展示**：该单词的不同用法
3. **视频片段**：该单词/短语在哪些视频中出现
- 点击搜索结果中的视频→**只播放包含该单词的那句台词**，单句循环
- 上划切换到下一个包含该单词的视频
- **免费用户**：每次展示 3 条语境视频；**付费用户**：无限 + 可按难度筛选

### 4.7 重遇弹幕
- 当前视频出现用户收藏过的单词时，视频上飞出弹幕
- 弹幕文案带记忆锚点，如"3 天前收藏 · 第 2 次重遇"
- 半透明背景，2-3 秒后消失，不影响观看
- 同词同 session 最多出现一次，有三档出现频率控制
- 点击弹幕可查看单词详细信息
- **重遇单词在字幕中高亮显示**：当前视频字幕中出现的收藏词会视觉高亮，一眼可见
- **弹幕完全免费不设限**——它是复习系统的前置钩子
- **付费点后移**：点击弹幕后的动作（加复习队列、加练、周报统计）需付费

### 4.8 微练习（6 种题型）

| # | 题型 | 说明 |
|---|------|------|
| 1 | 预测完型 3 选 1 | 给出句子空白，从 3 个选项中选正确答案 |
| 2 | 乱序单词拖拽拼句 | 将打乱顺序的单词拖拽到正确位置组成句子。点选后选项自动排序，缺交互设计待完善 |
| 3 | 8 词选 4 | 一段视频里 8 个词中选出听到的 4 个 |
| 4 | 选最合适的表达 | 看视频回答问题，选出最合适的表达 |
| 5 | 选视频主要内容 | 看视频选出主要内容。也是视频完整播放后自动插入微练习的默认第一题 |
| 6 | 选台词含义 | 看视频选出这句台词的含义 |

**触发方式**：
- 视频流页面有微练习按钮，用户主动点击进入
- **视频完整播放后自动插入微练习**（默认第一题为"这个视频描述了什么"）
- 入口做完不消失，可重复进入做题

**答题交互**：
- 提交答案后底部出现**「继续」按钮**，用户也可随时点 **「X」退出**答题
- 每道题答题结束后，**自动切换为"中英文字幕"模式显示**，方便学习
- 做完所有题型入口不消失，可重复做

**目标单词高亮**：
- 如果目标单词在题目中，题目需高亮目标单词
- 如果目标单词在答案中，完成答题后高亮显示
- 学习笔记中也同样高亮目标单词

**金币结算**：
- 每次答完题浮出金币结算通知气泡（+X 金币），不再弹独立结算页

**默认题量**：3 道简单题，做完展示"挑战更难的题"按钮
**播放速度**：×0.8 倍速，降低听力门槛
**与熟悉度打通**：出题范围包含用户收藏的单词，答题结果直接影响单词熟悉度等级
**计费方式**：按视频 session 计（1 个视频的练习 = 1 次），不按题目数量计
**免费用户**：每日 3 题，第 4 题起选项模糊，显示"升级解锁"
**付费用户**：无限

### 4.9 收藏夹 & 单词管理

收藏夹是打开 APP 的**第一屏**（视频流顶部标签）。

#### 熟悉度分级
**生词 → 眼熟 → 熟悉 → 精通**，四级递进。

#### 间隔重复机制
- 答对升一级（隔天才能再升级）
- 7～14 天不复习降一级
- 每天最多面对 5 个降级单词
- 取消了「批量变精通」和「批量取消精通」——精通只能通过答题推进

#### 每日守护
- 每天提醒用户守护 5 个即将降级的单词（做微练习题）
- 题型均衡分配 6 种题型
- 守护失败可在当前结算页直接重做，无需退出
- **仅付费用户可用**

#### 考一下
- 在收藏夹点开某个单词，针对该单词做微练习
- 支持一直做题直到题库耗尽
- ⚠️ 名称待调整

#### 导入辞书（v0.6 新增）
- 一键导入整本辞书（四六级、雅思、托福等，后台预置 1000+ 本）
- 导入时支持批量筛选，可一键剔除 A1/A2 基础词
- 收藏词与辞书词分组展示

#### 学习列表（v0.6 新增，月度目标制）
- 用户设定每月学习目标（如 100 词/月）
- 系统从待学习队列自动填充
- 目标：当月将所有单词推到「精通」
- 未完成的自动累计到下月
- 用户可随时调整目标
- 解决"单词债"问题——通过月度目标控制活跃词量，避免复习债务无限堆积

### 4.10 跟读评分
- 第三屏列出当前视频所有台词
- 点击任意一句→单句循环播放
- 用户点击"跟读"按钮→系统录音并评分
- 评分维度：发音准确度、语调、流利度（0-100 分 + 改进建议）
- 跟读模式：单句模式 / 连续模式 / 挑战模式（隐藏字幕）
- **跟读（听+跟念）免费，AI 评分仅付费**

### 4.11 会员中心（v0.6 新增）
- 新增会员中心页面
- 显示会员类型、有效期、已使用功能次数
- 视频上传功能迁移到手机端会员中心内（方便内容提供者/运营随时随地上传）

### 4.12 金币体系（v0.6 新增）
- 全局所有场景的金币增加数量规则
- 获取途径：每日签到、观看视频、完成微练习、分享视频、邀请好友
- 用途：兑换会员天数、临时解锁付费功能、兑换虚拟物品

### 4.13 学习数据报告
- 付费用户专属：掌握率、连续学习天数、词汇增长曲线
- 每周/每月推送

## 五、付费体系

### 5.1 核心策略

> **免费给"体验"，收费给"体系"**  
> 免费闭环：刷 + 听 + 查（让用户形成习惯）  
> 付费闭环：练 + 评 + 复习 + 进阶（让用户获得成长）

### 5.2 免费 vs 付费完整对照

| 功能 | 🆓 免费用户 | 👑 付费会员 |
|------|------------|------------|
| 刷视频 | ✅ 无限量 | — |
| 闪答 | ✅ 完全免费 | — |
| 纯英/中英/减压字幕 | ✅ 完全免费 | — |
| 延迟字幕 | ❌ | ✅ 可自定义延迟 2/3/5 秒 |
| 字典查词+框选 | ✅ 完全免费 | — |
| 收藏单词 | ✅ 不限量 | — |
| AI 教练 | 每日 3 次 | ✅ 无限次 |
| 学习笔记 | 前 2 条（后面毛玻璃） | ✅ 完整内容 |
| 跟读（听+跟念） | ✅ 免费 | — |
| 跟读 AI 评分 | ❌ | ✅ 评分+问题定位+建议 |
| 搜索语境视频 | 展示 3 条 | ✅ 无限+难度筛选 |
| 难度筛选视频 | ❌ 跟算法推荐 | ✅ 自选 1~5 星 |
| 每日守护+复习系统 | ❌ | ✅ 遗忘曲线/统计/复习计划 |
| 微练习 | 每日 3 题 | ✅ 无限 |
| 重遇弹幕 | ✅ 不限次 | ✅ 点击后可加复习队列/加练 |
| 学习数据报告 | ❌ | ✅ 掌握率/天数/增长曲线 |

### 5.3 定价方案

采用**单一订阅制、仅年付、不按模块拆分**（卖的是生态闭环整合价值，不是模块）。

**创始会员阶梯定价**：

| 阶段 | 新用户年卡 | 创始会员锁价 |
|------|-----------|-------------|
| 第一阶段（v0.6 上线） | 299 元/年 | 锁定 |
| 第二阶段 | 399 元/年 | 保持原价 |
| 第三阶段 | 498 元/年 | 保持原价 |

**内测用户专属价**：68 元/年（需 App Store 留真实评价）

**新用户转化**：Day 3-7 推送 3 天全功能体验（不在 Day 0 给，用户已有积累时价值感知更高），体验结束后转化。

**不做什么**：不限时终身会员、不推月付（v0.6 粘性不足以支撑月付留存，年付通过沉没成本倒逼习惯养成）、不搞点数包。

### 5.4 老用户续费
⏳ 第二阶段用户年费到期后，第三阶段涨价是否引起不满——**待决策**。候选方案：老用户锁价续费 / 阶梯缓冲价 / 提前预告+续费窗口。

## 六、设计语言

### 6.1 设计哲学

**沉浸式暗黑 + 橙色活力**。让用户感觉自己在刷短视频，不是在上课。像 TikTok/抖音一样让人上瘾，不像 Duolingo 一样卡通。

- **沉浸感优先**：纯黑底色（#000000），让视频和字幕成为绝对主角
- **橙色是信号灯**：只在 CTA、价格、进度完成、紧迫感时刻出现，不大面积铺色
- **触手可及**：所有关键操作（购买、收藏、跟读）在拇指热区内，单手可达
- **暗中发光**：用 glow/shadow 让关键元素在暗背景上"浮起来"
- **动效快而干脆**：150-400ms，按钮用 scale 反馈，卡片用 slide-up 入场

### 6.2 色彩系统

| 角色 | 色值 | 说明 |
|------|------|------|
| 主色 | #FF600F | 按钮、价格、核心强调 |
| 页面底色 | #000000 | 纯黑，不接受深灰替代 |
| 卡片背景 | #111111 | 一级卡片 |
| 嵌套背景 | #1A1A1A | 输入框、嵌套卡片 |
| 正文文字 | #AAAAAA | 不全部用白色 |
| 标题文字 | #FFFFFF | 核心信息 |
| 弱化文字 | #666666 | 禁用/辅助 |
| 金色 | #FFB800 | 星标、新功能标记 |
| 绿色 | #00C853 | 已完成/已上线 |
| 蓝色 | #448AFF | 信息/进行中 |

### 6.3 核心组件

- **主按钮 CTA**：橙色渐变 + glow 阴影 + 圆角 12px，每页最多 2 个
- **卡片**：深色背景 + 微弱边框，用阴影和间距分隔而非粗边框
- **Badge**：只用语义色（绿=完成、蓝=进行中、橙=热门、金=新增）
- **底部固定栏**：毛玻璃半透明（backdrop-filter: blur）+ safe-area 适配
- **价格展示**：大号橙色数字（36px/800 weight）+ 划线原价
- **图标**：统一线性 SVG（stroke-width 1.5），禁止 emoji 作为 UI 元素

### 6.4 页面叙事结构

1. Hero（身份标签 + 主标题 + 价格锚点）
2. 社会证明（数据统计）
3. 紧迫感（剩余名额/倒计时）
4. 核心功能（分卡片展示，带状态 Badge）
5. 路线图（时间线或阶段展示）
6. 价格对比（现在 vs 涨价后）
7. 信任保障（安全支付/无理由退/永久锁价）
8. FAQ（可展开）
9. 底部 CTA（fixed 按钮 + 价格提示）

## 九、附录：重要决策摘要

> 完整决策日志见 DECISIONS.md，此处仅摘要关键结论。

- **v0.6 开始收费**：创始会员阶梯定价 299→399→498，仅年付，不按模块拆分
- **免费/付费边界**：减压字幕免费（最强差异化不锁）、收藏不限量、重遇弹幕不限次（它是转化漏斗入口）
- **微练习与熟悉度打通**：答题直接影响单词等级，微练习成为记忆体系核心推进手段
- **导入辞书**：支持一键过滤 A1/A2 基础词，减少库存焦虑
- **学习列表月度目标制**：解决单词债问题，未完成自动累计下月
