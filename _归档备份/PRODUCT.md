## 一、是什么

**AI 英语听力学习 APP。**

把美剧、电影用 AI 剪辑成完整对话短片（每段 ≤2 分钟），以抖音信息流的形式刷视频，同时学英语听力。不是背单词 APP，不是刷题 APP，而是「在沉浸式刷剧中顺带把英语听力练出来」。

## 三、产品结构

每条视频有三屏：

| 屏 | 内容 |
|----|------|
| 第一屏：视频 | 视频播放 + 闪答 + 字幕 + 重遇弹幕，核心消费场景 |
| 第二屏：学习笔记 | AI 提炼的短语和知识点 |
| 第三屏：跟读 | 逐句台词，点击单句循环，跟读 + 评分 |

收藏夹是打开 APP 的**第一屏**（顶部标签），不是隐藏入口。

## 四、核心功能

### 4.1 短视频流
- 全屏沉浸，上下滑切视频，类抖音体验
- 视频难度 1～5 星（A1～C1），C1 以上统一 5 星
- 顶部分类标签：推荐 / 关注 / 发现 / 科技 / 热点 / 财经 / 娱乐

### 4.2 闪答
- 视频播放到指定位置，弹出迷雾遮罩
- 2 选 1，听到了哪个单词/短语
- 难度低，专注听就能答对
- 目的：保持专注度，防止「刷视频」走神
- 免费功能，无限次

### 4.3 字幕系统（4 种模式）

| 模式 | 说明 |
|------|------|
| 减压字幕 | 纯英文，但超纲词（超 A2）后附中文释义。差异化核心，**永远免费** |
| 中英字幕 | 双语同时显示 |
| 纯英文字幕 | 仅英文 |
| 延迟字幕 | 字幕延迟画面 2 秒后出现 |

### 4.4 全局 AI 教练
- 任意一句台词点「AI 一键解析」→ 弹出聊天窗口
- 结合上下文解答台词含义、语法、文化背景
- 支持追问，可连续对话
- ⚠️ **已知问题**：当前解答质量偏低，未达产品标准，高优先级待改进
- 免费用户：每日 3 次；付费用户：无限

### 4.5 全局字典 + 框选查词
- APP 内任意文字（台词、笔记、AI 教练对话）均可长按框选
- 弹出字典：释义、音标、词性、例句、难度等级
- 查词后可收藏单词到收藏夹
- 字典页可呼出 AI 教练继续追问
- **永远免费**

### 4.6 智能搜索
- 搜索结果三层：
  1. 字典（释义）
  2. 用法展示（不同用法、例句）
  3. 视频片段（该词/短语出现在哪些视频）
- 点击视频片段→跳到那句台词，单句循环播放
- 上划切换到下一个含该词的视频，同样单句循环
- 语境视频免费给 3 条（完全锁住会损害口碑传播）

### 4.7 重遇弹幕
- 当前视频出现用户**收藏过的单词**时，自动飞出弹幕
- 文案如：「3天前收藏 · 第2次重遇」
- 视觉做轻（半透明），同词同 session 最多一次，有三档控制防噪
- **完全免费不限次**——重遇是「火」，点击后的复习动作才是「灭火器」（付费）
- 点击弹幕可查看单词详情（释义、收藏时间、复习次数）

### 4.8 微练习（6 种题型）

| # | 题型 |
|---|------|
| 1 | 预测完型 3 选 1 |
| 2 | 乱序单词拖拽拼句 |
| 3 | 8 词选 4（从一段视频里选出听到的 4 个词）|
| 4 | 选最合适的表达（看视频+选项）|
| 5 | 选视频主要内容 |
| 6 | 选台词含义 |

**入口**（v0.6 起）：从随机插入改为用户**主动触发**，视频流页面新增按钮  
**题量**：默认 3 道简单题，做完可选「挑战更难的题」  
**速度**：微练习视频统一 ×0.8 倍速  
**与熟悉度打通**：答题结果直接影响单词熟悉度等级，微练习 = 复习推进器  
**出题范围**：包含用户收藏的单词  
免费用户每日 3 题，付费用户无限

出现场景：
- 视频流主动触发
- 收藏夹 → 每日守护
- 收藏夹 → 考一下
- 收藏夹 → 学习列表

### 4.9 收藏夹 & 单词管理

**熟悉度四级**：生词 → 眼熟 → 熟悉 → 精通  
精通只能通过答题推进（不能手动批量操作）

**间隔重复规则**：
- 答对升一级
- 隔天才能再升级
- 7～14 天不复习降一级
- 每天最多面对 5 个降级单词

**每日守护**：提醒守护 5 个即将降级的单词，做微练习，失败可在结算页直接重做（仅付费）

**考一下**：点开单个单词→做微练习，支持一直做到题库耗尽（功能名称待调整）

**导入辞书**（v0.6 新增）：
- 一键导入整本辞书（四六级、雅思、托福等，预置 1000+ 本）
- 导入时一键剔除 A1/A2 基础词
- 收藏词与辞书词分组展示

**学习队列 — 月度目标制**（v0.6 新增）：
- 用户设定每月学习目标（如 100 词/月）
- 系统自动从队列填充，目标是当月全部推到精通
- 未完成自动累计下月，用户可随时调整目标

### 4.10 跟读 & 评分
- 第三屏，逐句台词列表
- 点击任意句 → 单句循环播放
- 点跟读按钮 → 实时录音 → 评分（发音准确度、语调、流利度）
- 三种模式：单句 / 连续 / 挑战（隐藏字幕）
- **仅付费用户可用**

## 五、商业模式

### 核心理念
**免费给「体验」，收费给「体系」。**  
免费用户可无限刷视频、闪答、查词收藏，但复习系统、深度练习和数据报告需要付费。不按模块拆分收费，卖的是生态闭环整合，统一会员身份。

### 定价（年付，无月付，无终身会员）

| 阶段 | 新用户年卡 | 创始会员锁价 |
|------|-----------|-------------|
| 第一阶段（v0.6 上线） | **299 元/年** | 锁定 |
| 第二阶段 | 399 元/年 | 保持原价 |
| 第三阶段 | 498 元/年 | 保持原价 |

内测用户专属价：**68 元/年**（需在 App Store 留真实评价）  
新用户转化：Day 3-7 推送 3 天全功能体验（积累后体验，感知价值更高），体验结束引导付费

### 免费 vs 付费边界

| 功能 | 免费 | 付费 |
|------|------|------|
| 刷视频 | ✅ 无限 | — |
| 闪答 | ✅ 无限 | — |
| 字幕（含减压字幕） | ✅ 无限 | — |
| 查词 / 收藏（无容量限制） | ✅ 无限 | — |
| 重遇弹幕 | ✅ 无限 | — |
| 搜索语境视频 | ✅ 3 条 | 更多 |
| AI 教练 | ✅ 每日 3 次 | 无限 |
| 学习笔记 | ✅ 前 2 条 | 无限 |
| 微练习 | ✅ 每日 3 题 | 无限 |
| 每日守护 | ❌ | ✅ |
| 跟读评分 | ❌ | ✅ |
| 难度筛选 | ❌ | ✅ |
| 学习数据报告 | ❌ | ✅ |

## 六、v0.6 开发中变更汇总

> v0.6 = 头号听力第一个收费版本

1. **收费体系上线** — 创始会员，年付，阶梯定价
2. **微练习改版** — 主动触发、默认 3 题、×0.8 倍速、与熟悉度打通
3. **收藏夹升级** — 导入辞书（含过滤 A1/A2）、学习队列月度目标制、守护体验优化、取消批量变精通
4. **视频与交互升级** — 「更多」入口改版、单句循环 UI、跟读交互重做、闪答机制调整、「作者」改「上传者」、拖拽拼句 UI 优化
5. **会员中心** — 新增页面，视频上传功能迁移至此
6. **金币体系** — 设计全场景金币增减规则
7. **「考一下」** — 支持做到题库耗尽，名称待定
8. **取消** — ~~新用户引导（选目标、选水平）~~ v0.6 不做
