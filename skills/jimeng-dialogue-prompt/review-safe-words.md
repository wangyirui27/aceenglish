# 即梦审核安全词对照表

## 必须替换的中文词

| ❌ 危险词 | ✅ 安全替换 | 原因 |
|----------|-----------|------|
| 酒吧 | 演出场馆 / 俱乐部 / 排练室 | 含"酒"字，触发夜生活审核 |
| 酒红色 | 深红色 / 暗红色 / 勃艮第红 | 含"酒"字 |
| 酒吧街 | 商业街 / 夜市街 | 夜生活联想 |
| 纹身店 | 艺术工作室 | 纹身敏感 |
| 夜店 | 迪厅 / 娱乐场所（最好直接避免） | 夜生活 |
| 纹身 | 图案 / 花纹 / 身体彩绘 | 纹身在部分平台敏感 |
| 烟 / 抽烟 / 香烟 | （尽量不出现） | 烟草敏感 |
| 醉 / 喝醉 | （直接删除） | 酒精 |
| 低胸 / 性感 / 暴露 | （直接删除或换成风格描写） | 色情审核 |

## 安全的替代写法

| 场景需求 | ❌ 危险写法 | ✅ 安全写法 |
|----------|-----------|-----------|
| 爵士酒吧后台 | 纽约下东区一家老旧酒吧的后台 | 纽约一家老爵士乐俱乐部的后台休息室 |
| 红色裤子 | 酒红色高腰阔腿裤 | 深红色高腰阔腿裤 |
| 有纹身的角色 | 手臂有纹身 | 手臂有几何图案装饰 |
| 烟熏妆 | 烟熏妆眼影 | 深色眼影 / 浓郁眼妆 |
